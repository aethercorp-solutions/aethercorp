ðŸŒ³ What is a Tree?

A tree is a hierarchical data structure made up of nodes, with the following properties:

  • Each tree has a root node (top of the tree).
  • Each node can have zero or more child nodes.
  • Nodes with no children are called leaves.
  • There are no cycles (i.e., it's a connected acyclic graph).

Goal:

Traverse all nodes in the tree (preorder, inorder, or postorder for binary trees).

Python Code (Binary Tree)


class TreeNode:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

def preorder_traversal(root):
    if root is None:
        return []
    
    result = []
    def dfs(node):
        if node:
            result.append(node.val)      # Visit root
            dfs(node.left)              # Traverse left subtree
            dfs(node.right)             # Traverse right subtree
    
    dfs(root)
    return result

# Example usage:
#     1
#    / \
#   2   3
#  / \
# 4   5

root = TreeNode(1)
root.left = TreeNode(2)
root.right = TreeNode(3)
root.left.left = TreeNode(4)
root.left.right = TreeNode(5)

print(preorder_traversal(root))  # Output: [1, 2, 4, 5, 3]
