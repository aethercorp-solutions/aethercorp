.docx

A .docx file is actually a ZIP archive that contains a collection of XML files and other resources (like images and styles). This structure follows the Office Open XML (OOXML) standard.

ðŸ§¾ Key Files & Their Purpose

File Purpose
word/document.xml Main content of the Word document (paragraphs, text, tables, etc.)
word/styles.xml Defines styles used (fonts, spacing, etc.)
word/settings.xml Document-level settings
word/_rels/document.xml.rels Relationships (e.g., links to images or headers)
[Content_Types].xml Declares MIME types for parts of the package
docProps/core.xml Metadata (author, title, creation date, etc.)
docProps/app.xml Additional metadata (application name, total words, etc.)
