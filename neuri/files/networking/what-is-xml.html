eXtensible Markup Language XML

XML stands for eXtensible Markup Language. It's a markup language (like HTML) designed to store and transport data in a structured, readable format.

Key Features of XML:

  • Human-readable & machine-readable
  • Self-descriptive: The tags describe the data.
  • Hierarchical structure: Data is organized in a tree-like format.
  • Custom tags: Unlike HTML, XML doesn't have predefined tags. You create your own tags based on your needs.

Example of XML:


<person>
    <name>John Doe</name>
    <age>30</age>
    <email>john@example.com</email>
</person>

Common Uses:

  • Data exchange between systems (e.g., between APIs)
  • Configuration files (e.g., in Android or Java applications)
  • Web services (e.g., SOAP)
  • Storing data in a structured format
