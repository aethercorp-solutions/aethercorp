CAPTCHAs

CAPTCHAs (Completely Automated Public Turing test to tell Computers and Humans Apart) are tools used on websites to determine whether a user is a human or a bot. They help prevent automated abuse like spam, data scraping, or brute-force attacks.

Types of CAPTCHAs

  1. Text-based CAPTCHAs:
    • Show distorted letters/numbers that humans can read but bots struggle with.
  2. Image-based CAPTCHAs:
    • Ask users to select specific images (e.g., "Select all traffic lights").
  3. reCAPTCHA (Google):
    • v2: "I'm not a robot" checkbox, sometimes followed by image challenges.
    • v3: Invisible and score-based, runs in the background.
    • Enterprise: More advanced risk analysis.
  4. Audio CAPTCHAs:
    • Play distorted audio for users who may have visual impairments.
  5. Puzzle or Game CAPTCHAs:
    • Simple tasks like dragging a slider or rotating an object.
  6. Biometric/Behavioral CAPTCHAs:
    • Monitor mouse movement, typing patterns, or other behavior to infer humanity.

Why They Exist

  • Block bots from spamming forms.
  • Prevent credential stuffing.
  • Avoid ticket scalping or hoarding.
  • Protect online polls and registration forms.

Issues with CAPTCHAs

  • Accessibility: Can be hard for users with disabilities.
  • Frustration: Image CAPTCHAs can be slow and repetitive.
  • Privacy: Some, like reCAPTCHA, track user behavior and send data to third parties.

Here’s how you can implement a basic CAPTCHA on a website. I'll give you both a simple custom CAPTCHA and an example using Google reCAPTCHA, which is widely used and secure.

1. Custom CAPTCHA (Simple Text-Based)

Backend: Python (Flask) Example


from flask import Flask, render_template_string, request, session
import random
import string

app = Flask(__name__)
app.secret_key = 'your_secret_key'

def generate_captcha():
    return ''.join(random.choices(string.ascii_uppercase + string.digits, k=6))

@app.route('/', methods=['GET', 'POST'])
def index():
    if request.method == 'POST':
        user_input = request.form['captcha']
        if user_input == session.get('captcha'):
            return 'CAPTCHA passed ✅'
        else:
            return 'CAPTCHA failed ❌'

    captcha = generate_captcha()
    session['captcha'] = captcha
    return render_template_string('''
        <form method="POST">
            <label>Enter CAPTCHA: {{ captcha }}</label><br>
            <input type="text" name="captcha"<<br>
            <input type="submit">
        </form>
    ''', captcha=captcha)

if __name__ == '__main__':
    app.run(debug=True)

This example:

  • Generates a random CAPTCHA string
  • Stores it in session
  • Compares it on form submission

2. Google reCAPTCHA (v2 Checkbox)

Step-by-Step Setup

a. Register Your Site:

  • Go to Google reCAPTCHA Admin Console
  • Choose reCAPTCHA v2 (“I'm not a robot”)
  • Add your domain
  • Get Site Key and Secret Key

b. HTML (Frontend)


<form action="/submit" method="POST">
    <div class="g-recaptcha" data-sitekey="YOUR_SITE_KEY_HERE"></div>
    <br/>
    <input type="submit" value="Submit">
</form>

c. Backend: Python (Flask)


import requests
from flask import request

@app.route('/submit', methods=['POST'])
def submit():
    recaptcha_response = request.form['g-recaptcha-response']
    data = {
        'secret': 'YOUR_SECRET_KEY_HERE',
        'response': recaptcha_response
    }
    r = requests.post('https://www.google.com/recaptcha/api/siteverify', data=data)
    result = r.json()

    if result['success']:
        return 'CAPTCHA verified ✅'
    else:
        return 'CAPTCHA failed ❌'
