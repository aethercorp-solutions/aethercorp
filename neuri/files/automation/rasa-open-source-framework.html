Rasa is an open-source framework

Rasa is an open-source framework for building AI-powered conversational assistants (chatbots and voice assistants). It is especially popular among developers and enterprises for building contextual, flexible, and production-grade conversational systems.

ğŸ› ï¸ Core Concepts

  1. NLU (Natural Language Understanding)

    Rasa NLU is responsible for:

    • Intent classification: What the user wants to do (book_flight, greet, etc.)
    • Entity extraction: Extracting structured data (e.g., cities, dates)
    Example: 
    
# nlu.yml
- intent: book_flight
  examples: |
    - I want to book a flight to New York
    - Book me a ticket for tomorrow
  2. Core (Dialogue Management)

    Rasa Core handles the flow of the conversation using:

    • Stories: Show possible conversation paths
    • Rules: Define strict paths
    • Policies: ML models or rules that predict the next action

    Example:

    
# stories.yml
- story: book flight path
  steps:
    - intent: greet
    - action: utter_greet
    - intent: book_flight
    - action: flight_form
    - action: utter_confirm
  3. Actions

    Custom Python code that your bot executes, e.g., calling APIs or databases.

    
# actions.py
class ActionGetWeather(Action):
    def name(self):
        return "action_get_weather"

    def run(self, dispatcher, tracker, domain):
        city = tracker.get_slot("city")
        dispatcher.utter_message(text=f"The weather in {city} is sunny!")
        return []

Forms & Slots

Used to collect and track user input over multiple turns.
