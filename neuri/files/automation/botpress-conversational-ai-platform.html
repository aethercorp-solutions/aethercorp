Botpress

Botpress is a conversational AI platform that lets you build and deploy chatbots using a flow-based visual editor, Natural Language Understanding (NLU), and integrations with various channels (e.g., webchat, WhatsApp, Messenger, etc.).

üß† How Botpress Works (Overview)

1. Architecture Overview

Botpress has a modular architecture with these core components:

Component Description
Studio Visual editor to create conversations (flows, nodes, QnA, etc.)
NLU Engine Understands user input (intents, entities, slots)
Bot Engine Executes logic based on flows and actions
Channels Interfaces to connect your bot to Messenger, WhatsApp, Web, etc.
Hooks & Actions Custom server-side code to process logic or API calls
Database Stores conversations, user state, NLU models, etc.

2. Conversation Flow

  1. User sends a message (e.g., "I want to book a flight").
  2. Botpress:
    • Passes input to the NLU engine
    • Detects intent (book_flight) and entities (destination: Paris)
    • Triggers the corresponding flow
  3. The flow may:
    • Ask for missing information
    • Call an API (via custom action)
    • Store user data
    • Send a reply
  4. The conversation continues or ends based on the flow logic.
