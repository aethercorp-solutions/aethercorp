üß† 4. Build an AI Chatbot Using NLP/LLMs

Option 1: Use OpenAI API (like ChatGPT)

Requirements:

  • Python
  • openai library
  • OpenAI API key

Example:


import openai

openai.api_key = "your-api-key"

def ask_gpt(prompt):
    response = openai.ChatCompletion.create(
        model="gpt-4",  # Or gpt-3.5-turbo
        messages=[{"role": "user", "content": prompt}]
    )
    return response['choices'][0]['message']['content']

while True:
    user_input = input("You: ")
    if user_input.lower() in ["quit", "exit"]:
        break
    response = ask_gpt(user_input)
    print("Bot:", response)

This gives you a powerful conversational bot in just a few lines.

Simple Rule-Based Chatbot in Python


def chatbot():
    print("Hello! I'm your friendly chatbot. Type 'quit' to exit.")

    while True:
        user_input = input("You: ").lower()

        if "hello" in user_input or "hi" in user_input:
            print("Bot: Hello there! How can I help you today?")
        elif "how are you" in user_input:
            print("Bot: I'm just a program, but I'm doing great! Thanks for asking.")
        elif "name" in user_input:
            print("Bot: I don't have a name yet. What should I be called?")
        elif "help" in user_input:
            print("Bot: I can chat with you! Try saying 'hello', ask how I am, or just say 'bye' to exit.")
        elif "bye" in user_input or "quit" in user_input or "exit" in user_input:
            print("Bot: Goodbye! Have a great day!")
            break
        else:
            print("Bot: Sorry, I don't understand that. Can you try something else?")

if __name__ == "__main__":
    chatbot()

Great question! The line


if __name__ == "__main__":

is a very common Python idiom. Here‚Äôs what it means and why it‚Äôs used:

What is __name__?

  • In Python, every file (module) has a built-in variable called __name__.
  • When you run a Python script directly (e.g., python chatbot.py), Python sets __name__ to the string "__main__".
  • But if you import that script as a module into another Python file, __name__ is set to the module‚Äôs filename (without .py).

What does if __name__ == "__main__": do?

This line checks:

  • "Is this script being run directly?"

If yes, then the code inside this if block will execute.

If no (meaning the file was imported), the code inside will not run.

Why is it useful?

  • Run some code when executed directly (like starting your chatbot).
  • But avoid running that code if the script is imported somewhere else (to avoid unwanted side effects).

Example:


def chatbot():
    # chatbot code here

if __name__ == "__main__":
    chatbot()  # This only runs if the script is run directly

If you import this script in another file like this:


import chatbot_script  # assuming your file is chatbot_script.py

The chatbot() function won't run automatically, but you can still call chatbot_script.chatbot() manually.

  • if __name__ == "__main__": means "only run this code if this script is the main program."
  • It prevents code from running when the script is imported as a module.

I'll provide you with a simple example in Python using Flask (a lightweight web framework) that implements a basic rule-based chatbot API.

Rule-Based Chatbot API (Python + Flask)


from flask import Flask, request, jsonify

app = Flask(__name__)

# Define chatbot rules: keywords or phrases mapped to responses
rules = {
    "hello": "Hi there! How can I help you today?",
    "how are you": "I'm a bot, but I'm doing great! Thanks for asking.",
    "bye": "Goodbye! Have a nice day!",
    "help": "Sure! I can assist you with your queries. Ask me anything.",
}

def get_response(message):
    message = message.lower()
    for keyword, response in rules.items():
        if keyword in message:
            return response
    return "Sorry, I didn't understand that. Can you please rephrase?"

@app.route('/chatbot', methods=['POST'])
def chatbot():
    data = request.json
    user_message = data.get('message', '')
    
    if not user_message:
        return jsonify({"error": "No message provided"}), 400
    
    response_text = get_response(user_message)
    
    return jsonify({"response": response_text})

if __name__ == "__main__":
    app.run(debug=True)

How to use:

  1. Save the above code as app.py
  2. Install Flask if you don't have it: 
    
pip install flask
  3. Run the app: 
    
python app.py
  4. Make a POST request to http://127.0.0.1:5000/chatbot with a JSON body like: 
    
{
    "message": "Hello"
}
  5. The API will respond with: 
    
{
    "response": "Hi there! How can I help you today?"
}
