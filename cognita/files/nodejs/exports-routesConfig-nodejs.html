In Node.js (specifically with Express.js), the .use() method is used to:

ğŸ” 1. Middleware Usage

Middleware functions are functions that have access to the req, res, and next objects. They can:

  • Modify the request or response.
  • End the request-response cycle.
  • Call next() to pass control to the next middleware.

âœ… Example:

const app = express();

// Global middleware
app.use((req, res, next) => {
    console.log('Request URL:', req.originalUrl);
    next(); // Pass to the next middleware or route
});

This will run for every request.

ğŸ” 2. Serving Static Files

Serves static files from the public directory (like index.html, images, CSS, etc.)

ğŸ” 3. Mounting Routers

You use .use() to mount a route module (like in modularized apps):

âœ… Example:


const userRoutes = require('./routes/users.routes');

app.use('/api/users', userRoutes);

All routes in users.routes will be prefixed with /api/users.

So if you define router.get('/') inside users.routes.js, it will respond to GET /api/users.

  • app.use() matches all HTTP methods (GET, POST, PUT, etc.).
  • If you want to handle only specific methods, use app.get(), app.post(), etc.
