Introduction to Node.js

Node.js is a runtime environment that allows you to run JavaScript code outside the browser — primarily used to build server-side applications.

What is Node.js?

  • Node.js is built on the V8 JavaScript engine (the same engine that powers Google Chrome).
  • Created by Ryan Dahl in 2009.
  • It uses an event-driven, non-blocking I/O model — which makes it lightweight and efficient.
  • Ideal for building real-time applications, like chat apps, APIs, and streaming services.

Key Features

Feature Description
Asynchronous & Event-Driven Handles many connections at once without blocking the main thread.
Fast Execution Built on V8 engine, compiles JavaScript to native machine code.
Single-Threaded Uses a single-threaded event loop to handle multiple clients.
NPM (Node Package Manager) A massive ecosystem of open-source libraries and tools.
Cross-platform Runs on Windows, macOS, and Linux.

Common Use Cases

  • RESTful APIs and web services
  • Real-time chat applications (e.g., using Socket.io)
  • Data streaming applications
  • Microservices architecture
  • Command-line tools

What is NPM?

NPM (Node Package Manager) is the default package manager for Node.js.

  • Helps manage libraries/dependencies.
  • Example: npm install express installs the Express framework.

Example: A Simple HTTP Server


const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, Node.js!');
});

server.listen(3000, () => {
  console.log('Server running on http://localhost:3000/');
});

This code sets up a basic server that listens on port 3000 and responds with "Hello, Node.js!"

How to Get Started

  1. nstall Node.js from https://nodejs.org
  2. Open terminal or command prompt.
  3. Run node to enter the REPL (Read-Eval-Print Loop).
  4. Create a .js file and run it using node filename.js.
