Modern JavaScript features (ES6 and beyond)

Modern JavaScript (ES6 and beyond) includes many powerful features that improve code readability, maintainability, and performance. Here’s a breakdown of the most notable features introduced from ES6 (2015) onward:

ES6 (ECMAScript 2015)

  1. let and const 
    
let x = 10;
const y = 20;
    • let is block-scoped.
    • const is also block-scoped and cannot be reassigned.
  2. Arrow Functions 
    
const add = (a, b) => a + b;
    • Concise syntax.
    • Lexically binds this.
  3. Template Literals 
    
const name = "Jane";
console.log(`Hello, ${name}!`);
  4. Destructuring 
    
const [a, b] = [1, 2];
const {name, age} = {name: "John", age: 30};
  5. Default Parameters 
    
function greet(name = "Guest") {
    console.log(`Hello, ${name}`);
}
  6. Spread and Rest Operators 
    
const nums = [1, 2, 3];
const more = [...nums, 4, 5];

function sum(...args) {
  return args.reduce((acc, val) => acc + val, 0);
}
  7. Classes 
    
class Person {
    constructor(name) {
        this.name = name;
    }
    
    greet() {
        console.log(`Hi, I'm ${this.name}`);
    }
}
  8. Modules 
    
// Export
export const PI = 3.14;

// Import
import { PI } from './math.js';
  9. Promises 
    
fetch('api/data')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

    ES7–ES13 (2016–2022) Highlights

  10. Async/Await (ES8) 
    
async function fetchData() {
    try {
        const res = await fetch('api/data');
        const data = await res.json();
        console.log(data);
    } catch (err) {
        console.error(err);
    }
}
  11. Object.entries(), Object.values(), Object.fromEntries() (ES8/ES10) 
    
Object.entries({a: 1, b: 2}); // [['a', 1], ['b', 2]]
  12. Optional Chaining (?.) and Nullish Coalescing (??) (ES11)< 
    
const user = {};
console.log(user?.profile?.name); // undefined
console.log(user.name ?? "Anonymous"); // "Anonymous"
  13. Array.prototype.includes (ES7) 
    
[1, 2, 3].includes(2); // true
  14. String.prototype.padStart / padEnd (ES8) 
    
'5'.padStart(3, '0'); // "005"

    ES2023+ (New & Emerging Features)

  15. Top-Level Await 
    
const res = await fetch('api/data'); // Works at module top-level
  16. Logical Assignment Operators 
    
a ||= 10; // a = a || 10
b &&= 20; // b = b && 20
  17. Array at() Method 
    
const arr = [10, 20, 30];
arr.at(-1); // 30
  18. Promise.allSettled 
    
Promise.allSettled([p1, p2, p3]).then(results => {
    // Each result is { status: 'fulfilled' | 'rejected', value | reason }
});
