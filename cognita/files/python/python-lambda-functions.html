üêç Python Lambda Functions

A lambda function in Python is a small anonymous function defined using the lambda keyword. It's often used when you need a simple function for a short period of time ‚Äî typically as an argument to functions like map(), filter(), or sorted().

üîπ Syntax:


lambda arguments: expression

üîπ Example:


add = lambda x, y: x + y
print(add(2, 3))  # Output: 5

üîπ Common Use:


numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)  # Output: [1, 4, 9, 16, 25]

üåê JavaScript Arrow Functions (Similar to Lambda)

In JavaScript, there isn't a lambda keyword, but arrow functions (=>) serve a similar purpose ‚Äî they define anonymous or short functions.

üîπ Syntax:


const add = (x, y) => x + y;
console.log(add(2, 3));  // Output: 5

üîπ Common Use:


const numbers = [1, 2, 3, 4, 5];
const squared = numbers.map(x => x ** 2);
console.log(squared);  // Output: [1, 4, 9, 16, 25]
