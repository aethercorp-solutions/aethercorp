ü™Ñ 3. Using pathlib (Recommended Modern Way)

pathlib is cleaner and object-oriented.


from pathlib import Path

# Create a Path object
p = Path("example.txt")

# Check if file exists
print(p.exists())

# Get file name and extension
print(p.name, p.suffix)

# List files in a folder
for file in Path(".").iterdir():
    print(file)

# Create directory
Path("data").mkdir(exist_ok=True)

# Read and write text files easily
p.write_text("Hello World!")
print(p.read_text())

üì¶ 4. Copy, Move, and Delete Files (shutil)


import shutil

# Copy file
shutil.copy("source.txt", "backup.txt")

# Move file
shutil.move("backup.txt", "folder/backup.txt")

# Remove directory with all contents
shutil.rmtree("folder")

üîç 5. Searching Files (glob)


import glob

# Find all .txt files
for file in glob.glob("*.txt"):
    print(file)

# Recursive search
for file in glob.glob("**/*.py", recursive=True):
    print(file)

üß† 6. Example: Practical Script

Here‚Äôs an example of scanning and organizing files by type:


from pathlib import Path
import shutil

downloads = Path("/Users/paul/Downloads")
organized = downloads / "organized"

for file in downloads.iterdir():
    if file.is_file():
        ext = file.suffix[1:] or "others"
        folder = organized / ext
        folder.mkdir(parents=True, exist_ok=True)
        shutil.move(str(file), str(folder / file.name))

print("Files organized successfully!")

This script:

  • Creates folders by file type (e.g., pdf, jpg, others)
  • Moves files from your Downloads folder into those folders automatically.
