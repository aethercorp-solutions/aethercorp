Key difference between map object(iterator) and list.

Great question—this is a core Python concept.

🔑 Key Difference

Feature map object (iterator) list
Type Iterator (lazy) Concrete collection
Evaluation Lazy (on demand) Eager (immediate)
Memory Low (doesn't store all values) Higher (stores all values)
Reusability ❌ One-time use ✅ Can reuse anytime
Indexing ❌ Not allowed ✅ Allowed (list[0])
