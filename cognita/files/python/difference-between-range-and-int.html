Difference between range and int

In Python, range and int (integer) are completely different types with different purposes:

🔢 int (Integer)

  • Represents a single whole number
  • Can be positive, negative, or zero
  • Used for calculations

Example:


x = 5
y = -10
z = 0

You can do math with it:


print(x + 2)  # 7

🔁 range

  • Represents a sequence of numbers
  • Commonly used in loops
  • Doesn’t store all numbers in memory (efficient)

Example:


r = range(5)

This represents:


0, 1, 2, 3, 4

Used in loops:


for i in range(5):
    print(i)

⚖️ Key Differences

Feature int range
Type Single number Sequence of numbers
Usage Math operations Iteration (loops)
Example 5 range(5)
Memory Stores value Generates values on demand

💡 Simple Analogy

  • int = one number (like a single house 🏠)
  • range = a list of numbers (like a street of houses 🏘️)

Great—let’s break down range(start, stop, step) clearly 👇

🔁 Full Syntax


range(start, stop, step)

📌 Meaning:

  • start → where the sequence begins
  • stop → where it ends (NOT included)
  • step → how much to increment (or decrement)

✅ Example 1: Basic range


for i in range(1, 6):
    print(i)

Output:


1 2 3 4 5

👉 Starts at 1, stops before 6

⏩ Example 2: Using step


for i in range(0, 10, 2):
    print(i)

Output:


0 2 4 6 8

👉 Jumps by 2 each time

⏪ Example 3: Negative step (reverse)


for i in range(5, 0, -1):
    print(i)

Output:


5 4 3 2 1

👉 Counts backwards

🧠 Important Rules

1. stop is always excluded


list(range(3))

Output:


[0, 1, 2]

2. Default values

If you omit parameters:


range(5)

Same as:


range(0, 5, 1)

3. Step cannot be zero ❌


range(1, 10, 0)  # ERROR

🔍 Visual Understanding

Think of it like:


range(start, stop, step)

➡️ Start here → move by step → stop before limit

💡 Pro Tip

If you want to actually see the numbers:


print(list(range(1, 10, 3)))

Output:


[1, 4, 7]
