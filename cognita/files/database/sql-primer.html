What is SQL?

SQL (Structured Query Language) is the standard language used to communicate with databases. You use it to:

  • Store data
  • Retrieve data
  • Update data
  • Delete data

Think of it as the language that tells a database what to do.

SQL Primer: Basic Concepts

1. Database and Tables

  • A database is like a digital filing cabinet.
  • A table is like a drawer in that cabinet where data is organized into rows and columns.

Example table: Users

ID Name Email
1 Paul paul@email.com
2 Alice alice@email.com

2. Basic SQL Statements

a. SELECT – Read data


SELECT Name, Email
FROM Users;
  • SELECT chooses the columns
  • FROM chooses the table

b. INSERT – Add data


INSERT INTO Users (Name, Email)
VALUES ('John', 'john@email.com');

c. UPDATE – Modify data


UPDATE Users
SET Email = 'newemail@email.com'
WHERE ID = 1;

d. DELETE – Remove data


DELETE FROM Users
WHERE ID = 2;

3. Filtering Data

  • WHERE helps you filter rows: 
    
SELECT * 
FROM Users
WHERE Name = 'Paul';
  • You can use conditions like =, >, <, >=, <=, LIKE (for patterns), IN, etc.

4. Sorting Data


SELECT * 
FROM Users
ORDER BY Name ASC; -- ASC = ascending, DESC = descending

5. Joining Tables

If you have multiple tables, you can combine them with JOIN:


SELECT Orders.ID, Users.Name
FROM Orders
JOIN Users ON Orders.UserID = Users.ID;

6. Aggregating Data

  • Use COUNT, SUM, AVG, MIN, MAX:

SELECT COUNT(*) AS TotalUsers
FROM Users;

7. Other Useful Clauses

  • GROUP BY – groups rows with the same value
  • HAVING – filters groups
  • LIMIT – limits number of rows returned
  • DISTINCT – returns unique values only

COALESCE(Nickname, "") AS Nickname


COALESCE(Nickname, "") AS Nickname

What does this do?

  • COALESCE() checks for NULL values
  • If Nickname is NULL, it returns an empty string ""
  • If Nickname has a value, it returns that value

Why is this useful?

Without it, you might see NULL in the results, which can:

  • Look messy
  • Cause issues in apps or reports

Using AS Nickname just renames the output column to Nickname.
