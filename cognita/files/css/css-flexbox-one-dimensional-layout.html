üß† What is Flexbox?

Flexbox is a one-dimensional layout model in CSS used to align and distribute space among items in a container, even when their size is unknown or dynamic.

üß± 1. Basic Structure


<div class="flex-container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>
            

.flex-container {
    display: flex;
    flex-direction: row;       /* row | row-reverse | column | column-reverse */
    justify-content: center;   /* flex-start | center | flex-end | space-between | space-around | space-evenly */
    align-items: center;       /* flex-start | center | flex-end | stretch | baseline */
    flex-wrap: wrap;           /* nowrap (default), wrap */
    gap: 10px;
}
    
.item {
    padding: 10px;
    border: 1px solid #666;
}

üìç On the Container (display: flex)

Property What it Does
flex-direction Direction of items (row/column)
justify-content Main-axis alignment (left to right in a row)
align-items Cross-axis alignment (top to bottom in a row)
flex-wrap Allows wrapping (nowrap/wrap)
gap Space between items

üì¶ 3. On Flex Items

Property What it Does
flex Shorthand for flex-grow, flex-shrink, and flex-basis
flex-grow Item can grow to fill space
flex-shrink Item can shrink if needed
flex-basis Initial size before growing/shrinking
align-self Override align-items per item
order Change visual order 

.item {
    flex: 1; /* means flex-grow: 1, flex-shrink: 1, flex-basis: 0 */
    }
    
.item:first-child {
    order: 2;
}
    
.item:last-child {
    align-self: flex-end;
}

Sidebar

Navigation links or other fixed content goes here.

Main Content

This content area expands to fill the remaining space. You can put your main app content, articles, dashboards, or anything else here. 


.layout {
    display: flex;
    height: 100vh; /* Full viewport height */
}

.sidebar {
    flex: 0 0 200px;
    background-color: #2c3e50;

.content {
    flex: 1;
    background-color: #ecf0f1;

Your CSS sets up a basic flexbox layout with a fixed-width sidebar and a content area that expands to fill the remaining space. Here's a breakdown of what each part does:

‚úÖ .layout


.layout {
    display: flex;
}
  • This makes the container a flex container (default direction is row), so its direct children (like .sidebar and .content) will be laid out horizontally.

‚úÖ .sidebar


.sidebar {
    flex: 0 0 200px; /* fixed width */
}
  • flex: 0 0 200px is shorthand for:
    • flex-grow: 0 ‚Üí don't grow.
    • flex-shrink: 0 ‚Üí don't shrink.
    • flex-basis: 200px ‚Üí start with 200px width.
  • So the sidebar will always stay exactly 200px wide.

‚úÖ .content


.content {
    flex: 1; /* take remaining space */
}
  • flex: 1 is shorthand for flex: 1 1 0%, meaning:
    • flex-grow: 1 ‚Üí take all remaining space.
    • flex-shrink: 1 ‚Üí shrink if needed.
    • flex-basis: 0% ‚Üí initial size is 0, but grow to fill space.
  • This makes .content expand to fill the rest of the row, after the sidebar.

‚úÖ Summary

This layout will produce a two-column structure:

  • Left: .sidebar with a fixed 200px width.
  • Right: .content dynamically resizing to fill the rest.
