CSS Rules

CSS (Cascading Style Sheets) rules are used to style HTML elements on a web page. A CSS rule consists of a selector and a declaration block.

ðŸ”¹ Basic Syntax of a CSS Rule


selector {
    property: value;
    property: value;
}

ðŸ”¹ Example


p {
    color: blue;
    font-size: 16px;
}

This CSS rule styles all <p> elements to have blue text and a font size of 16 pixels.

ðŸ”¹ CSS Rule Components

Component Description
Selector Specifies which HTML element(s) the rule applies to
Property The style attribute you want to change (e.g. color, margin)
Value The value assigned to the property (e.g. red, 20px)
Selector Type Example Description
Universal * Applies to all elements
Element h1 Targets all <h1> tags

In CSS, layout rules define how elements are arranged on the page â€” including their position, size, spacing, and how they interact with other elements. These rules are essential for building responsive and well-structured web pages.

Here are the key properties and techniques used for layout:

1. Display

Controls the box type and layout behavior of an element.


display: block | inline | inline-block | flex | grid | none;
  • block â€“ Starts on a new line, takes full width (e.g. <div>)
  • inline â€“ Flows within text (e.g. <span>)
  • flex â€“ Enables Flexbox layout
  • grid â€“ Enables Grid layout
  • none â€“ Hides the element

2. Position

Controls how an element is positioned in the document.


position: static | relative | absolute | fixed | sticky;
  • static â€“ Default (normal document flow)
  • relative â€“ Positioned relative to itself
  • absolute â€“ Positioned relative to the nearest positioned ancestor
  • fixed â€“ Positioned relative to the viewport
  • sticky â€“ Sticks to a position while scrolling

3. Flexbox Layout

Used for one-dimensional layouts (row or column).


display: flex;
justify-content: center;
align-items: center;
flex-direction: row | column;

âœ… Example:

    display: flex;

4. Grid Layout

Used for two-dimensional layouts (rows and columns).


display: grid;
grid-template-columns: repeat(3, 1fr);
grid-gap: 20px;

âœ… Example:

    display: grid;

5. Margin and Padding

Used for spacing outside and inside elements.


margin: 20px;
padding: 10px;
  • margin: space outside the border
  • padding: space inside the border

6. Width and Height

Defines the size of elements.


width: 100%;
max-width: 1200px;
height: auto;

7. Box-Sizing

Controls how padding and border are included in element dimensions.


box-sizing: border-box;

This is commonly used in resets:


* {
    box-sizing: border-box;
}

8. Float (Legacy Layout)

Still used occasionally, but mostly replaced by Flexbox and Grid.


float: left;

You often need to clear floats:


.clearfix::after {
    content: "";
    display: table;
    clear: both;
}
