CSS display property

In CSS, the display property determines how an element is displayed on the page. It is one of the most important layout-related properties in CSS.

üîπ Common display Values

Value Description
block The element is displayed as a block (takes up full width). Example: <div>, <p>.
inline The element is displayed inline (doesn't break the line). Example: <span>, <a>.
inline-block Like inline, but allows setting width and height.
none The element is not displayed (hidden, no space taken).
flex Enables Flexbox layout. Used for flexible layouts.
grid Enables Grid layout.
inline-flex Flex layout but behaves like inline.
inline-grid Grid layout but behaves like inline.
contents Makes the element disappear but keeps its children visible.
list-item Used for list items like <li> shows a bullet or number.

    display: flex;
    justify-content: space-between;
