Axios

Axios is a popular JavaScript library used to make HTTP requests from:

  • Browsers (client-side)
  • Node.js (server-side)

It‚Äôs often used with frontend frameworks like React, Vue, or Angular, but it works in any JavaScript environment.

üîß Why Use Axios?

  • Simpler syntax than fetch
  • Automatically transforms JSON data
  • Supports request & response interceptors
  • Supports async/await
  • Handles timeouts, headers, and more out of the box

import axios from 'axios';

// GET request
axios.get('https://api.example.com/data')
  .then(response => {
    console.log(response.data);
  })
  .catch(error => {
    console.error(error);
  });

// POST request
axios.post('https://api.example.com/data', {
  name: 'John',
  age: 30
})
.then(response => {
  console.log(response.data);
})
.catch(error => {
  console.error(error);
});

async function fetchData() {

        console.log(response.data);
    } catch (error) {
        console.error(error);
    }
}
