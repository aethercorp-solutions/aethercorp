StyleSheet API

Styling in React Native uses a concept similar to CSS, but it‚Äôs implemented via JavaScript objects using the StyleSheet API. Additionally, responsive design in React Native often requires different techniques compared to traditional web development due to the wide variety of device sizes and resolutions.

üîπ 1. Basic Styling with StyleSheet


import { StyleSheet, Text, View } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
        <Text style={styles.text}>Hello World</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
    alignItems: 'center',
    justifyContent: 'center',
  },
  text: {
    color: 'blue',
    fontSize: 20,
  },
});

Note: Styles are not inherited by default (unlike web CSS).

üîπ 2. Inline Styling

You can apply styles directly:


<Text style={{ fontSize: 18, color: 'red' }}>Inline style</Text>
            
Use inline styles for quick overrides or dynamic styling.

üîπ 3. Combining Styles

You can pass an array of styles:


<Text style={[styles.text, { color: 'green' }]}>Combined style</Text>

Later styles override earlier ones.

üîπ 4. Responsive Design Techniques

React Native doesn‚Äôt support media queries directly, so here are alternatives:

‚úÖ A. Dimensions API

const { width, height } = Dimensions.get('window');

if (width < 400) {
  // Small screen logic
}

‚úÖ B. Percentage-Based Layouts

You can use % in certain properties:


<View style={{ width: '80%', height: '50%' }} />

‚úÖ C. Flexbox Layout

React Native uses Flexbox by default for layout:


container: {
    flex: 1,
    flexDirection: 'column', // or 'row'
    justifyContent: 'center',
    alignItems: 'center',
}

‚úÖ D. Platform-Specific Styling


import { Platform } from 'react-native';

const styles = StyleSheet.create({
  text: {
    ...Platform.select({
      ios: { fontSize: 20 },
      android: { fontSize: 18 },
    }),
  },
});

‚úÖ E. Hooks for Responsiveness: useWindowDimensions()


import { useWindowDimensions } from 'react-native';

const MyComponent = () => {
  const { width } = useWindowDimensions();

  return (
    <View style={{ padding: width > 400 ? 20 : 10 }}>
        <Text>Responsive Text</Text>
    </View>
  );
};

üîπ 5. Best Practices

  • Use StyleSheet.create() for performance and readability.
  • Use flex layouts and percentage-based sizing.
  • Avoid fixed dimensions unless absolutely necessary.
  • Test on multiple screen sizes and orientations.
  • Consider dark mode using useColorScheme().

In React Native, StyleSheet is a way to define styles similar to CSS, but in JavaScript.

Instead of writing:


.myClass {
    color: red;
}

You write:


const styles = StyleSheet.create({
    myClass: {
        color: 'red',
    },
});

Then apply it to your component:


<Text style={styles.myClass}>Hello</Text>

‚úÖ What is Responsive Design?

Responsive design means your app adjusts automatically to different screen sizes (small phones, large tablets, etc.).

So instead of your app looking good only on an iPhone 13, it also looks good on:

  • Android phones
  • iPads
  • Foldable phones
  • Any other screen size

üìê Example

Let‚Äôs say you want a button that looks good on all screens.


import { Dimensions } from 'react-native';

const { width } = Dimensions.get('window');

const styles = StyleSheet.create({
  button: {
    width: width * 0.8, // 80% of screen width
    padding: 10,
    backgroundColor: 'blue',
  },
});

This way, the button takes up 80% of the screen width ‚Äî no matter how big or small the device is.

‚úÖ What is Flexbox?

Flexbox is a layout system (built-in to React Native) that lets you easily arrange elements.


container: {
    flex: 1,
    flexDirection: 'row',       // arrange horizontally
    justifyContent: 'center',   // center items horizontally
    alignItems: 'center',       // center items vertically
}

It replaces the need for using margin, padding, or exact positioning in many cases.
