React Hooks

React Hooks are functions that let you use state and other React features without writing a class. Here’s a quick overview of the most commonly used ones:

1. useState

Purpose: Add state to functional components.


import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      Clicked {count} times
    </button>
  );
}

useState(initialValue) returns an array: [state, setState].

2. useEffect

Purpose: Side effects like data fetching, DOM manipulation, subscriptions, etc.


import { useEffect, useState } from 'react';

function Timer() {
  const [seconds, setSeconds] = useState(0);

  useEffect(() => {
    const interval = setInterval(() => setSeconds(s => s + 1), 1000);

    return () => clearInterval(interval); // Cleanup on unmount
  }, []); // Empty dependency = run once on mount
}
  • Runs after render.
  • Dependencies array controls when it runs again.

3. useRef

Purpose: Access DOM elements or persist values across renders without triggering re-renders.


import { useRef, useEffect } from 'react';

function FocusInput() {
  const inputRef = useRef(null);

  useEffect(() => {
    inputRef.current.focus(); // Direct DOM manipulation
  }, []);

  return <input ref={inputRef} />;
}
  • Holds mutable value: ref.current.
  • Doesn’t cause re-renders when updated.

4. useMemo

Purpose: Memoize expensive computations.


import { useMemo } from 'react';

function ExpensiveComponent({ number }) {
  const double = useMemo(() => {
    console.log("Computing...");
    return number * 2;
  }, [number]);

  return <div>{double}</div>;
}
  • Recalculates only when dependencies change.
  • Useful for optimizing performance.

5. useCallback

Purpose: Memoize functions so they aren’t re-created on every render.


import { useCallback, useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  const increment = useCallback(() => {
    setCount(c => c + 1);
  }, []);

  return <button onClick={increment}>Count: {count}</button>;
}

useCallback(fn, deps) is the same as useMemo(() => fn, deps).

Summary Table

Hook Purpose Triggers Re-render?
useState Add state Yes
useEffect Perform side effects No (by itself)
useRef Mutable reference (DOM/value) No
useMemo Memoize computed values No
useCallback Memoize functions No
