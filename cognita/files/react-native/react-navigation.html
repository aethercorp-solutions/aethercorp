React Navigation

React Navigation is a popular library used for routing and navigation in React Native applications. It helps you move between different screens and manage navigation state across your app.

ðŸ”§ Installation

You can install React Navigation using npm or yarn:


npm install @react-navigation/native

Then install the required dependencies:


npx expo install react-native-screens react-native-safe-area-context
npm install react-native-gesture-handler react-native-reanimated

If you're using Expo, some of these might already be included.

Finally, install a specific navigator (e.g., Stack Navigator):


npm install @react-navigation/native-stack

ðŸ“± Basic Usage (Stack Navigator)

Here's a basic example using createNativeStackNavigator:


// App.js
import * as React from 'react';
import { NavigationContainer } from '@react-navigation/native';
import { createNativeStackNavigator } from '@react-navigation/native-stack';
import HomeScreen from './screens/HomeScreen';
import DetailsScreen from './screens/DetailsScreen';

const Stack = createNativeStackNavigator();

export default function App() {
  return (
    <NavigationContainer>
        <Stack.Navigator initialRouteName="Home">
            <Stack.Screen name="Home" component={HomeScreen} />
            <Stack.Screen name="Details" component={DetailsScreen} />
        </Stack.Navigator>
    </NavigationContainer>
  );
}

And a sample screen:


// screens/HomeScreen.js
import React from 'react';
import { Button, View, Text } from 'react-native';

const HomeScreen = ({ navigation }) => (
  <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
    <Text>Home Screen</Text>
        <Button
      title="Go to Details"
      onPress={() => navigation.navigate('Details')}
    />
</View>
);

export default HomeScreen;

React Navigation offers several types of navigators:

Navigator Package Use Case
Stack Navigator @react-navigation/native-stack Standard screen-to-screen navigation
Tab Navigator @react-navigation/bottom-tabs Bottom tab navigation
Drawer Navigator @react-navigation/drawer Sidebar menu navigation
Material Top Tabs @react-navigation/material-top-tabs Swipeable tabs
