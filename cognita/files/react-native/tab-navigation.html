Tab Navigation lets users switch between top-level sections of your app using tabs (usually at the bottom or top).

Think of tabs as:

‚ÄúDifferent rooms in the same house‚Äù

Each tab represents an independent feature area, not a step in a process.

Use Tab Navigation when:

‚úÖ 1. Sections Are Peer-Level (No Hierarchy)

  • Home
  • Search
  • Notifications
  • Profile

‚úÖ 2. Frequent Switching Is Expected

Users move back and forth often.

  • Messaging apps
  • Social apps
  • Finance dashboards
  • Contact management apps (relevant to TryCastanet)
