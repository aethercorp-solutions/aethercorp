TypeScript is a superset of JavaScript that adds static typing. This means you can catch many errors at compile time instead of runtime.

üìù Step 2: Your First TypeScript File

üìö Interfaces

Interfaces define object shapes.


interface Person {
    name: string;
    age: number;
}
    
function printPerson(person: Person) {
console.log(`${person.name} is ${person.age} years old.`);
}

printPerson({ name: "Bob", age: 25 });

üß™ Bonus: TypeScript + Node.js Example

  1. Install Node.js types: 
    
npm init -y
npm install typescript @types/node --save-dev
  2. Add a script to package.json: 
    
"scripts": {
    "build": "tsc",
    "start": "node dist/index.js"
}
  3. Run: 
    
npm run build
npm start

The node command is used to run JavaScript code outside of a web browser using Node.js ‚Äî a JavaScript runtime built on Chrome's V8 engine.

Basic Usage:


node <filename>

For example:


node app.js

This command runs the app.js file with Node.js.

What Node.js Enables:

Before Node.js, JavaScript was mostly used only in browsers. Node.js lets you:

  • Run JavaScript on the server (backend)
  • Build command-line tools
  • Create APIs and full web applications
  • Work with files, databases, and networks

Examples:

  1. Run a simple script 
    
echo "console.log('Hello, world!')" > hello.js
node hello.js
  2. Output:

node
