StyleSheet

In React Native, StyleSheet is used to define styles for components in a structured and performant way. It’s similar to CSS but optimized for mobile. Here’s a comprehensive guide:

1. Basic Usage


import React from 'react';
import { View, Text, StyleSheet } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
      <Text style={styles.title}>Hello, React Native!</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,                  // fills the screen
    justifyContent: 'center', // vertically center
    alignItems: 'center',     // horizontally center
    backgroundColor: '#f5f5f5',
  },
  title: {
    fontSize: 24,
    fontWeight: 'bold',
    color: '#333',
  },
});

2. Key Points About StyleSheet

  1. Performance:

    Using StyleSheet.create() ensures styles are immutable and sent only once over the bridge to native code, improving performance.

  2. Flexbox Layout:

    React Native uses Flexbox by default for layout.

    
container: {
    flex: 1,
    flexDirection: 'row', // column (default) or row
    justifyContent: 'space-between',
    alignItems: 'center',
}
  3. Units:

    There are no px, em, or % units. All values are in density-independent pixels (dp).

  4. Colors:

    Accepts color names ('red'), hex codes ('#ff0000'), rgba ('rgba(255,0,0,0.5)').

3. Conditional / Dynamic Styles

You can combine or dynamically apply styles:


<Text style={[styles.text, isActive && styles.activeText]}>
    Dynamic Style
</Text>
    
    const styles = StyleSheet.create({
    text: { fontSize: 16, color: 'gray' },
    activeText: { color: 'blue', fontWeight: 'bold' },
});

4. In-line Styles

For very simple or dynamic styles, you can write them inline:


<View style={{ padding: 10, backgroundColor: isActive ? 'blue' : 'gray' }}>
    <Text>Hello</Text>
</View>  >

But for complex apps, StyleSheet.create is preferred for maintainability.

5. Platform-Specific Styles

React Native supports Platform-specific styles:


import { Platform, StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  container: {
    padding: Platform.OS === 'ios' ? 20 : 10,
  },
});

6. Responsive / Percentage-Based Layouts


container: {
    width: '80%',  // percentage works
    height: '50%',
}

For full responsiveness, you may combine with Dimensions:


import { Dimensions } from 'react-native';
const { width, height } = Dimensions.get('window');

container: {
  width: width * 0.8,
  height: height * 0.5,
}

✅ Best Practices

  1. Use StyleSheet.create() for all static styles.
  2. Keep styles in a separate file for larger projects.
  3. Use Flexbox instead of absolute positioning whenever possible.
  4. Use arrays for conditional styles.
  5. Avoid inline styles for performance-critical components.
