datetimepicker

For older React Native versions, you usually need to install a specific version of @react-native-community/datetimepicker that matches your RN version. The latest version may not work with older RN projects.

1️⃣ Install for older React Native

Example:


npm install @react-native-community/datetimepicker@7.0.0

or


yarn add @react-native-community/datetimepicker@7.0.0

Then (for older RN versions):


react-native link @react-native-community/datetimepicker

2️⃣ Version compatibility

Use the version based on your React Native version:

React Native Datetimepicker
RN ≥ 0.73 7.6.3+
RN 0.70 – 0.72 7.x
RN < 0.70 ≤ 7.0.0

This library replaced the old core components (DatePickerIOS, DatePickerAndroid, TimePickerAndroid) that were removed from React Native.

3️⃣ Example usage


import DateTimePicker from '@react-native-community/datetimepicker';

<DateTimePicker
  value={new Date()}
  mode="date"
  display="default"
  onChange={(event, date) => console.log(date)}
/>

4️⃣ For very old React Native (<0.60)

Autolinking does not work, so you must:


react-native link @react-native-community/datetimepicker
