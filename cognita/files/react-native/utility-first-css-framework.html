utility-first CSS Framework

A utility-first CSS framework is a type of CSS framework that provides low-level utility classes that can be used to build custom designs directly in your HTML, rather than relying on pre-designed components or large, opinionated class structures.

Instead of defining styles in separate CSS files and giving them semantic class names (like .card or .button-primary), you compose designs using small, single-purpose utility classes directly in the markup.

Each class does one thing:

  • bg-blue-500: background color
  • text-white: white text
  • p-4: padding
  • rounded: border radius
  • shadow-lg: large shadow

  • No naming conflicts: No need to invent unique class names.
  • Great for design systems: Encourages consistency and scalability.
