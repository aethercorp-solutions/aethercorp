webpack.config.js


const path = require('path');

module.exports = {
  entry: './src/index.js', // entry point of your app
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'), // output folder
  },
  module: {
    rules: [
      {
        test: /\.css$/, // handle CSS files
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.js$/, // transpile JS with Babel
        exclude: /node_modules/,
        use: 'babel-loader',
      },
    ],
  },
  mode: 'development', // or 'production' for minified output
};
  1. Entry Point 
    
entry: './src/index.js',
    • Webpack starts building the dependency graph from this file.
  2. Output 
    
output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
},
    • Bundled output is written as bundle.js in the dist/ directory.

  3. Loaders (under module.rules)

    a. CSS Loader

    
{
    test: /\.css$/,
    use: ['style-loader', 'css-loader'],
}
    • css-loader: Interprets @import and url() like import/require() and resolves them.
    • style-loader: Injects CSS into the DOM via <style> tags.
    b. JavaScript with Babel 
    
{
    test: /\.js$/,              // Target all files ending in .js
    exclude: /node_modules/,    // Ignore files in node_modules folder
    use: 'babel-loader',        // Use babel-loader to transpile the JS files
}
    • babel-loader: Transpiles ES6+/JSX into ES5 using Babel.
    • exclude: /node_modules/: Speeds up the build process by ignoring external dependencies.

    ðŸ”§ What it does:

    • It matches .js files with the test regular expression.
    • It excludes node_modules (since you usually don't want to transpile third-party packages).
    • It uses babel-loader, which allows Babel to transpile ES6+ JavaScript down to ES5 so it can run in older browsers.

    When Webpack encounters a .js file, babel-loader:

    1. Passes the file to Babel.
    2. Babel applies the configured presets/plugins (like @babel/preset-env).
    3. The transformed (older) JavaScript is then bundled by Webpack.
