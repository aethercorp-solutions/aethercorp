React Native

React Native is an open-source framework developed by Meta (formerly Facebook) that allows you to build mobile applications using JavaScript and React. With React Native, you can write code once and deploy it on both iOS and Android, which makes it a popular choice for cross-platform mobile app development.

Key Concepts

  1. JavaScript + React
    • React Native uses JavaScript as its programming language.
    • It follows the React paradigm of building UI using components.
  2. Native Components
    • React Native bridges the gap between JavaScript code and native platform code.
    • UI components like View, Text, and Button map to their native equivalents on iOS and Android.
  3. Cross-Platform
    • You can share most of your codebase between iOS and Android.
    • Platform-specific code can be written when needed.

Example: A Simple React Native App


import React from 'react';
import { View, Text, StyleSheet } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
        <Text>Hello, React Native!</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
});

Core Components

Component Description
View Container for layout
Text Displays text
Image Shows images
TextInput Input field
ScrollView Scrollable content container
StyleSheet Style abstraction for layout

Why Use React Native?

  • Code reusability across platforms
  • Hot Reloading for faster development
  • Active community & ecosystem
  • Integration with native modules when needed

Getting Started

To create a new React Native app, you can use the Expo CLI or React Native CLI:


npx create-expo-app MyApp
# or
npx react-native init MyApp
