When NOT to Use Stack Navigation

‚ùå Primary App Sections

Use Tab or Drawer, not Stack.

‚ùå Quick global switches

Stack is linear, not lateral.

‚ùå Complex state jumps

If screens don‚Äôt have a parent-child relationship, stack may confuse users.

Stack vs Other Navigators (Quick Comparison)

Navigator Best For
Stack Drill-down, auth, forms, details
Tab Top-level sections
Drawer App-wide navigation
Modal Stack Temporary tasks
Native Stack High performance, iOS-like UX

Native Stack vs JS Stack (Important)

Since you‚Äôre performance-focused, prefer:


import { createNativeStackNavigator } from '@react-navigation/native-stack'

Use Native Stack when:

  • Heavy screens
  • iOS polish matters
  • You want UIKit-level animations

Use JS Stack when:

  • You need fully custom animations
  • Gesture control beyond native limits

Real Production Example (Clean Pattern)

    </Stack.Navigator>
  )
}
