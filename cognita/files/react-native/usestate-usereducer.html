useState and useReducer

In React Native (just like in React for the web), useState and useReducer are two important React Hooks used for managing state in function components.

  const [count, setCount] = useState(0);

  return (

âœ… When to use:

  • For complex state logic (e.g., multiple related values).
  • When the next state depends on the previous state.
  • For better scalability and testability.

ğŸ“Œ Syntax:

ğŸ“¦ Example:


import React, { useReducer } from 'react';
import { View, Text, Button } from 'react-native';

const initialState = { count: 0 };

function reducer(state, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      return state;
  }
}

const Counter = () => {
  const [state, dispatch] = useReducer(reducer, initialState);

  return (
    <View>
        <Text>Count: {state.count}</Text>
        <Button title="Increase" onPress={() => dispatch({ type: 'increment' })} />
        <Button title="Decrease" onPress={() => dispatch({ type: 'decrement' })} />
    </View>
  );
};

export default Counter;
