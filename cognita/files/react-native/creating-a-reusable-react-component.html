2. Deploy to GitHub

A. Create a GitHub Repository

  1. Go toGitHub
  2. Create a new repository, e.g., my-button-component

B. Push Code to GitHub

In your terminal:


cd my-button-component
git init
git remote add origin https://github.com/your-username/my-button-component.git
git add .
git commit -m "Initial commit with reusable Button component"
git push -u origin master
