What is Tailwind CSS?

Tailwind CSS is a utility-first CSS framework for rapidly building custom designs. Instead of writing custom CSS, you compose styles by applying classes directly in your HTML.

âœ… Option 2: Install Tailwind in a Project (for real development)

1. Set up your project


mkdir my-tailwind-project
cd my-tailwind-project
npm init -y
npm install -D tailwindcss
npx tailwindcss init

2. Create your folder structure


mkdir src
touch src/index.html
mkdir styles
touch styles/tailwind.css

3. Configure Tailwind

In tailwind.config.js, update:


module.exports = {
    content: ["./src/**/*.{html,js}"],
    theme: {
        extend: {},
    },
    plugins: [],
}

4. Add Tailwind directives to CSS

In styles/tailwind.css:


@tailwind base;
@tailwind components;
@tailwind utilities;

5. Build your CSS

Install PostCSS and autoprefixer:


npm install -D postcss autoprefixer
npx tailwindcss -i ./styles/tailwind.css -o ./dist/output.css --watch

6. Use in HTML

Create src/index.html:


<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Tailwind Setup</title>
  <link href="../dist/output.css" rel="stylesheet">
</head>
<body class="bg-gray-100 p-10">

  <h1 class="text-3xl font-bold text-indigo-700">Hello from Tailwind!</h1>

</body>
</html>
