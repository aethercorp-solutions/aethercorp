Set up and use Storybook in a React Native project

Here's a straightforward tutorial to set up and use Storybook in a React Native project. Storybook is a great tool for developing and testing UI components in isolation.

Step-by-Step: Setup Storybook in React Native

Prerequisites

Make sure you already have:

  • A React Native project (react-native init MyApp or via Expo)
  • Node.js and npm/yarn installed

3. üì≤ Link Storybook with App Entry

Modify App.js to toggle between your app and Storybook:


import React, { useState } from 'react';
import { View, Text, Button } from 'react-native';
import StorybookUIRoot from './storybook';

export default function App() {
  const [showStorybook, setShowStorybook] = useState(false);

  if (showStorybook) {
    return ;
  }

  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text>Welcome to your app</Text>
      <Button title="Open Storybook" onPress={() => setShowStorybook(true)} />
    </View>
  );
}

Or for development only, you can set an environment variable or use __DEV__ to conditionally load Storybook.
