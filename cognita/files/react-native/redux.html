Redux with React Native

If you're a beginner in Redux with React Native, here's a straightforward guide to help you get started.

üîÅ What is Redux?

Redux is a state management library that helps you manage and share state across your entire app ‚Äî useful when props drilling gets out of hand.

üß± Redux Basics (Concepts You Need to Know)

  • Store: Holds the global state.
  • Actions: Plain JS objects describing what happened.
  • Reducers: Functions that update the state based on actions.
  • Dispatch: Sends an action to the store.
  • Provider: Makes the store accessible to the app.

üì¶ Installing Redux in a React Native App

Step 1: Create or use a React Native app

If you don‚Äôt have one:


npx react-native init ReduxExample
cd ReduxExample

Step 2: Install dependencies


npm install @reduxjs/toolkit react-redux

üõ†Ô∏è Setting Up Redux (with Redux Toolkit)

Redux Toolkit simplifies Redux by reducing boilerplate.

1. Create a slice (your feature/state logic)

Create a file: features/counterSlice.js


import { createSlice } from '@reduxjs/toolkit';

export const counterSlice = createSlice({
  name: 'counter',
  initialState: {
    value: 0,
  },
  reducers: {
    increment: state => { state.value += 1; },
    decrement: state => { state.value -= 1; },
  },
});

export const { increment, decrement } = counterSlice.actions;
export default counterSlice.reducer;

2. Create the store

Create a file: store.js


import { configureStore } from '@reduxjs/toolkit';
import counterReducer from './features/counterSlice';

export const store = configureStore({
  reducer: {
    counter: counterReducer,
  },
});

3. Provide the store to your app

In App.js:


import React from 'react';
import { Provider } from 'react-redux';
import { store } from './store';
import CounterScreen from './CounterScreen'; // your component

export default function App() {
  return (
    <Provider store={store}>
        <CounterScreen />
    </Provider>
  );
}

4. Use Redux state and dispatch in a component

Create CounterScreen.js:


import React from 'react';
import { View, Text, Button } from 'react-native';
import { useSelector, useDispatch } from 'react-redux';
import { increment, decrement } from './features/counterSlice';

export default function CounterScreen() {
  const count = useSelector(state => state.counter.value);
  const dispatch = useDispatch();

  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
        <Text style={{ fontSize: 30 }}>{count}</Text>
        <Button title="Increment" onPress={() => dispatch(increment())} />
        <Button title="Decrement" onPress={() => dispatch(decrement())} />
    </View>
  );
}

Redux is a state management library commonly used with JavaScript and React applications. It provides a centralized store to manage and share application state in a predictable and scalable way.

üîß When to Use Redux

Use Redux when:

  • Your app has complex state logic.
  • You need to share state across many components.
  • Debugging or testing state changes is important.
  • You want predictable state changes with undo/redo, logging, etc.

üß† Core Concepts

  1. Store ‚Äì The single source of truth (an object tree).
  2. Actions ‚Äì Plain JS objects that describe what happened.
  3. Reducers ‚Äì Functions that describe how the state changes.
  4. Dispatch ‚Äì Method to send actions to the reducer.
  5. Selectors ‚Äì Functions to retrieve specific parts of state.

üöÄ Basic Implementation (Vanilla JS + Redux)

1. Install Redux (if using outside React):


npm install redux

2. Create a Reducer


// counterReducer.js
const initialState = { count: 0 };

function counterReducer(state = initialState, action) {
  switch (action.type) {
    case "INCREMENT":
      return { count: state.count + 1 };
    case "DECREMENT":
      return { count: state.count - 1 };
    default:
      return state;
  }
}

export default counterReducer;

3. Create Store


// store.js
import { createStore } from 'redux';
import counterReducer from './counterReducer';

const store = createStore(counterReducer);

export default store;

4. Dispatch Actions


import store from './store';

store.subscribe(() => console.log(store.getState()));

store.dispatch({ type: "INCREMENT" }); // count: 1
store.dispatch({ type: "DECREMENT" }); // count: 0

üõ† Redux with React (Using react-redux)

1. Install React Redux

2. Setup Provider in Root

ReactDOM.render(
  <Provider store={store}>
    <App />
  </Provider>,

3. Connect Component


// Counter.js
import React from 'react';
import { useSelector, useDispatch } from 'react-redux';

function Counter() {
  const count = useSelector((state) => state.count);
  const dispatch = useDispatch();

  return (

        <h2>Count: {count}</h2>
        <button onClick={() => dispatch({ type: 'INCREMENT' })}>+</button>
        <button onClick={() => dispatch({ type: 'DECREMENT' })}>-</button>
    </div>
  );
}

export default Counter;

‚ö° Redux Toolkit (Modern Redux)

Redux Toolkit simplifies Redux by:

  • Reducing boilerplate
  • Adding built-in dev tools support

Great! Let's walk through how to implement Redux using the modern, recommended approach: Redux Toolkit (RTK), including handling asynchronous actions like API calls (using createAsyncThunk).

üöÄ What Is Redux Toolkit?

Redux Toolkit (RTK) is the official, opinionated way to write Redux logic. It simplifies store setup, reduces boilerplate, and includes powerful tools like:

  • createSlice ‚Äì Generates actions + reducers together
  • configureStore ‚Äì Sets up the store with good defaults
  • createAsyncThunk ‚Äì Handles async logic (like API calls)

üß± 1. Install Redux Toolkit and React Redux


npm install @reduxjs/toolkit react-redux

üì¶ 2. Set Up the Store


// store.js
import { configureStore } from '@reduxjs/toolkit';
import counterReducer from './features/counterSlice';

const store = configureStore({
  reducer: {
    counter: counterReducer,
  },
});

export default store;

‚úÇÔ∏è 3. Create a Slice (State + Reducers + Actions in one)


// features/counterSlice.js
import { createSlice, createAsyncThunk } from '@reduxjs/toolkit';

// Optional: Async example
export const fetchCount = createAsyncThunk(
  'counter/fetchCount',
  async () => {
    const res = await fetch('https://api.countapi.xyz/hit/example.com/visits');
    const data = await res.json();
    return data.value; // assuming it returns { value: number }
  }
);

const counterSlice = createSlice({
  name: 'counter',
  initialState: {
    value: 0,
    loading: false,
  },
  reducers: {
    increment: (state) => { state.value += 1; },
    decrement: (state) => { state.value -= 1; },
    reset: (state) => { state.value = 0; },
  },
  extraReducers: (builder) => {
    builder
      .addCase(fetchCount.pending, (state) => {
        state.loading = true;
      })
      .addCase(fetchCount.fulfilled, (state, action) => {
        state.value = action.payload;
        state.loading = false;
      })
      .addCase(fetchCount.rejected, (state) => {
        state.loading = false;
      });
  },
});

export const { increment, decrement, reset } = counterSlice.actions;
export default counterSlice.reducer;

üß© 5. Use Redux in Components


// Counter.js
import React from 'react';
import { useSelector, useDispatch } from 'react-redux';
import { increment, decrement, reset, fetchCount } from './features/counterSlice';

function Counter() {
  const count = useSelector((state) => state.counter.value);
  const loading = useSelector((state) => state.counter.loading);
  const dispatch = useDispatch();

  return (
    <div>
        <h2>Count: {loading ? 'Loading...' : count}</h2>
        <button onClick={() => dispatch(increment())}>+</button>
        <button onClick={() => dispatch(decrement())}>-</button>
        <button onClick={() => dispatch(reset())}>Reset</button>
        <br /><br />
        <button onClick={() => dispatch(fetchCount())}>
            Fetch from API
        </button>
        </div>
  );
}

export default Counter;
