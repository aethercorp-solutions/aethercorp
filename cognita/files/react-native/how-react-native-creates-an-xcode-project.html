  1. You run this command

  2. The CLI copies iOS template files

    React Native CLI internally pulls from this template:

    
node_modules/react-native/template/ios

    These template files include:

    • MyApp.xcodeproj (Xcode project)
    • MyApp.xcworkspace (CocoaPods workspace)
    • AppDelegate.mm
    • Info.plist
    • Podfile
    • bridging config for Obj-C/Swift
    • default assets & build configs
  3. The CLI replaces template tokens

    Anything with placeholders such as:

    • __PROJECT_NAME__
    • __ORG_IDENTIFIER__

    ‚Ä¶gets replaced with the actual values:

com.mycompany.myapp

    The result becomes your final ios/ folder.
