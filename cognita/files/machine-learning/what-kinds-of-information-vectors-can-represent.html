What kinds of information vectors can represent

A vector is a way to represent information as an ordered list of numbers. Each number captures one feature (attribute, measurement, or aspect) of whatever you’re describing. Vectors show up everywhere because they’re compact, precise, and easy to work with mathematically.

Below I’ll walk through what kinds of information vectors can represent, with clear examples and detailed explanations.

1. Physical quantities (magnitude + direction)

This is the classic meaning of a vector in physics.

Example: Velocity of a car

A car moving northeast at 60 km/h can be represented as:


v = (42.4, 42.4)

Explanation

  • The first value = speed in the x-direction (east)
  • The second value = speed in the y-direction (north)
  • Together, they fully describe how fast and in what direction the car moves

Why a vector?

  • Speed alone (60 km/h) loses direction
  • The vector preserves both direction and magnitude

Great question — this example is exactly where vectors start to feel real instead of abstract. Let’s break it down slowly and cleanly.

What does “60 km/h toward northeast” mean?

“Northeast” means the direction is halfway between east and north.

So the car is:

  • moving east
  • and north
  • by the same amount

That’s the key idea.

Step 1: Set up directions (x and y axes)

We choose a coordinate system:

  • x-axis → East
  • y-axis → North

So any velocity vector looks like:


v = (vx​, vy)

where:

  • vx = speed toward the east
  • vy = speed toward the north

Step 2: Split the 60 km/h into components

The total speed is 60 km/h, but that speed is shared between east and north.

Because northeast is a 45° angle, the east and north components are equal.

We use trigonometry:


vx = 60 cos(45deg)
vy = 60 cos(45deg)

And since:

cos(45 deg) = sin(45 deg) = sqrt(2)/2 == 0.707

Step 3: Do the math

vx = 60 x 0.707 == 42.4 vy = 60 x 0.707 == 42.4

Final vector

v = (42.4, 42.4)

This means:

  • The car moves 42.4 km/h east
  • and 42.4 km/h north
  • at the same time

Together, those two components combine to give:

  • total speed = 60 km/h
  • direction = northeast

Quick check (why this works)

If you recombine the components:


speed = sqrt(42.4^2 + 42.4^2) == sqrt(3600) == 60

So the math checks out perfectly.

Intuitive way to think about it

Imagine:

  • 1 hour passes
  • the car travels 42.4 km east
  • and 42.4 km north

Draw that as a right triangle — the diagonal path is exactly 60 km toward northeast.

2. Position or location in space

Vectors can describe where something is.

Example: Position on a map

If a point is 3 km east and 4 km north of a city center:


p = (3, 4)

Explanation

  • Each coordinate is a feature of position
  • In 3D space, you’d use three numbers: (x, y, z)
  • GPS coordinates are essentially vectors

3. Objects with multiple attributes (data representation)

Any object described by multiple properties can be turned into a vector.

Example: Student information

Suppose we track:

  • Age
  • Years of education
  • GPA

Student = (20, 14, 3.8)

Explanation

  • Each dimension represents one attribute
  • The vector becomes a compact numeric summary of the student
  • Many students → many vectors → a dataset

This is the foundation of data science and machine learning.

4. Images as vectors

Computers represent images using vectors.

Example: Grayscale image

A 28×28 pixel image (like handwritten digits) becomes:


x = (x1, x2, ..., x784)

Explanation

  • Each pixel intensity is a number (0–255)
  • All pixel values are stacked into one long vector
  • Machine learning models “see” images as vectors, not pictures

Color images just use more numbers (e.g., RGB values).

5. Text and language (word vectors / embeddings)

Words and sentences can also be vectors.

Example: Word embeddings

The word “king” might be represented as:


k = (0.21, -0.34, 0.89, ...)

Explanation

  • Each number captures a hidden linguistic feature
  • Similar words have similar vectors
  • Famous result:

king - man + woman == queen

This is how search engines, translators, and chatbots understand meaning.

6. Preferences and behavior

Vectors can model human choices.

Example: Movie preferences

A user rates genres from 0 to 5:


v = (Action = 5, Comedy = 3, Drama = 1)

Explanation

  • Each dimension = a preference
  • Recommendation systems compare vectors to find similar users
  • Netflix, Spotify, and Amazon rely heavily on this idea

7. Time-based measurements

Vectors can store measurements taken over time.

Example: Daily temperatures over a week


t = (28, 30, 29, 31, 33, 32, 30)

Explanation

  • Each element corresponds to a specific day
  • Useful for forecasting, pattern detection, and analysis

Why vectors are so powerful

Vectors let us:

  • Represent complex things numerically
  • Compare objects using distance or similarity
  • Perform calculations (dot product, norms, projections)
  • Feed real-world information into algorithms

In short:

If something can be described using numbers, it can be represented as a vector.
