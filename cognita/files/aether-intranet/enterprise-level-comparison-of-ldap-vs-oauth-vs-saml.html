2. LDAP (Lightweight Directory Access Protocol)

2.1 What LDAP Is

LDAP is a directory protocol, not just authentication.

It stores:

  • Users
  • Passwords
  • Groups
  • Organizational structure

Example:


cn=John Doe,ou=HR,dc=company,dc=com

4. SAML (Security Assertion Markup Language)

4.1 What SAML Is

SAML is an XML-based SSO protocol designed for enterprises.

It uses assertions instead of tokens.

4.4 Strengths

‚úî Very secure

‚úî Widely adopted in enterprises

‚úî Mature compliance support

‚úî Strong federation model

4.5 Weaknesses

‚ùå XML is heavy

‚ùå Hard to debug

‚ùå Not mobile-friendly

‚ùå Slow compared to OAuth

‚ùå Legacy feel

4.6 Best Use Cases

  • Legacy enterprise portals
  • Government systems
  • Large corporations
  • Existing SAML infrastructure

4.7 Example SAML Assertion (Simplified)


<saml:Assertion>
    <saml:Subject>user@company.com</saml:Subject>
    <saml:Attribute>HR</saml:Attribute>
</saml:Assertion>
