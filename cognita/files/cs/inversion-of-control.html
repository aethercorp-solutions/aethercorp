Inversion of Control (IoC)

Inversion of Control (IoC) is a software design principle where the control of creating objects or managing the flow of a program is inverted ‚Äî moved away from your code and handled externally.

üöÄ Simple Explanation

Normally, your code calls functions, creates objects, and controls the sequencing.

With IoC, your code does NOT control everything anymore. Instead, a framework or external component controls when your code runs and what objects it receives.

üëç Why It Matters

IoC helps make code:

  • More modular
  • Easier to test
  • More reusable
  • Less dependent on concrete implementations

üîß Example Without IoC


// You control object creation
const service = new EmailService();
const controller = new UserController(service);

controller.registerUser();

Here, UserController decides what implementation of EmailService to use.

üîÑ Example With IoC (Dependency Injection)


class UserController {
    constructor(emailService) {
        this.emailService = emailService;
    }
}
    
const controller = new UserController(new EmailService());

But with full IoC (using a container):


const controller = IoCContainer.resolve("UserController");

Here, the IoC container decides:

  • how to create UserController
  • what EmailService instance to inject

    Your code simply receives ready-to-use objects.

üß© IoC in Context of React Native or Node.js

React Native

  • Hooks provide inversion of control: the framework calls your code at the right time.
  • Navigation libraries (e.g., React Navigation) decide screen flow; you only provide components.
  • Event listeners: the framework triggers your callback, not you.

Node.js

  • Express routes: the framework calls your handler when a request arrives.
  • Serverless/Lambda: AWS controls execution; you only provide the handler function.
  • Dependency injection libs like:
    • awilix
    • inversify
    • tsyringe

üî• Ultra-Simple Analogy

Without IoC:

You drive the car.

With IoC:

You sit in the car, but someone else decides when you drive and where you go.
