2. Diffing in React (Virtual DOM)

React keeps a Virtual DOM (a lightweight copy of the UI tree).

Steps:

  1. Render new Virtual DOM
  2. Diff it against the previous one
  3. Apply the minimum changes to the real DOM

Example:


<p>Hello</p>

‚Üí


<p>Hello, Paul</p>

Diff result:

  • Text node changed
  • No need to recreate the <p> element
