A simple Python quine (and breakdown)

You showed this earlier; here it is again and explained:


s = 's = %r\nprint(s %% s)'
print(s % s)

Breakdown:

  1. s is a string that contains a format placeholder %r (which prints a Python representation including quotes).
  2. s %% s inside the string is written as %% so that when the outer string is formatted, a single % goes to the inner formatting of print.
  3. print(s % s) replaces %r with the representation of s, producing the complete source.

Step-by-step:

  1. s contains the template of the whole program (including print(s %% s)).
  2. s % s substitutes the representation of s into the template.
  3. The result is exactly the two lines of source, which print outputs.

1. Classic Python quine (safe)

Copy‚Äìpaste and run locally. It prints exactly its own source.


# quine.py
s = 's = %r\nprint(s %% s)'
print(s % s)

How it works (step-by-step)

  1. s is a string that literally contains a format placeholder %r that will be replaced by the representation of s itself.
  2. print(s % s) substitutes s into itself, producing the full source text, then prints it.
  3. The trick: the string includes the code necessary to recreate itself.

Test

python quine.py ‚Üí output identical to the file contents.

4. Simple safe self-copying program (Python) ‚Äî makes a timestamped backup of itself

This is not a quine (it reads its file) but is a benign self-copyer useful for auditing.


# backup_self.py
import shutil, datetime, pathlib

src = pathlib.Path(__file__)
ts = datetime.datetime.utcnow().strftime("%Y%m%dT%H%M%SZ")
dst = src.with_name(f"{src.stem}_backup_{ts}{src.suffix}")

shutil.copyfile(src, dst)
print(f"Backed up {src} -> {dst}")

Notes

  • This reads the file and writes a new copy in the same directory. It does not propagate across machines.
  • Good for automated versioned backups; combine with logging.

Implementation tips & debugging

  • For quines, avoid reading files (that would be cheating for the pure puzzle).
  • Use the language‚Äôs string quoting and formatting carefully ‚Äî inside/outside quotes matter.
  • To verify correctness: compare program output to exact file contents (byte-for-byte).
    • Example (Unix): python quine.py > out.txt && cmp quine.py out.txt
  • If output differs, inspect escaping and whitespace ‚Äî quines are sensitive.
