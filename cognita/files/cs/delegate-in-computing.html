üß© Usage

  1. Callbacks

    Delegates are often used for callback functions, where one piece of code signals or notifies another after a task is completed.

    Example:

    When a file download completes, a delegate can call a user-defined method like OnDownloadComplete().

  2. Event Handling

    Delegates are heavily used in event-driven programming, such as GUI frameworks or asynchronous operations.

    Example: In C#, buttons use delegates to trigger events:

    
button.Click += OnButtonClick;

    Here, OnButtonClick is a method assigned to the delegate that executes when the event occurs.

  3. Functional Programming

    Delegates are the foundation for lambda expressions and anonymous functions ‚Äî used to pass behavior as parameters.

    Example:

    
List<int> numbers = new List<int> {1, 2, 3, 4};
    numbers.ForEach(x => Console.WriteLine(x));

    Here, the lambda x => Console.WriteLine(x) is a delegate.
