Basics of Mathematical Reasoning

Mathematical reasoning is the process of using logic to solve problems and prove statements. It's a fundamental part of mathematics that allows us to go beyond guesswork and intuition to develop clear, logical conclusions.

🔹 1. Statements

A statement (or proposition) is a sentence that is either true or false, but not both.

Examples:

  • "2 + 2 = 4" → ✅ True statement
  • "5 is an even number" → ❌ False statement
  • "x + 3" → ❌ Not a statement (it's an expression)

🔹 2. Types of Reasoning

➤ Deductive Reasoning

  • Starts with general rules and applies them to specific cases.
  • Used in proofs.

Example:


All humans are mortal.
Socrates is a human.
⇒ Socrates is mortal.

➤ Inductive Reasoning

  • Uses specific examples to find general patterns.
  • Not always 100% certain.

Example:


2 is even, 4 is even, 6 is even...
⇒ All even numbers are divisible by 2.

🔹 3. Logical Connectives

Symbol Meaning Example
∧ AND P ∧ Q is true if both are true
∨ OR P ∨ Q is true if at least one is true
¬ NOT ¬P is true if P is false
⇒ Implies P ⇒ Q means "if P then Q"
⇔ If and only if P ⇔ Q means "P is true exactly when Q is true"

🔹 4. Quantifiers

➤ Universal Quantifier ( ∀ )

Means "for all"


∀x ∈ ℝ, x² ≥ 0
(For all real numbers, the square is non-negative.)

➤ Existential Quantifier ( ∃ )

Means "there exists"


∃x ∈ ℤ such that x² = 4
(There exists an integer x such that x² = 4.)

🔹 5. Contrapositive, Converse, Inverse

Let the original statement be:

P ⇒ Q (If P then Q)

Form Statement
Converse Q ⇒ P
Inverse ¬P ⇒ ¬Q
Contrapositive ¬Q ⇒ ¬P (Logically equivalent to original)

🔹 6. Proof Techniques

➤ Direct Proof

Use known facts and logic to prove a statement step-by-step.

➤ Contrapositive Proof

Instead of proving P ⇒ Q, prove ¬Q ⇒ ¬P.

➤ Proof by Contradiction

Assume the opposite of what you want to prove, and show that this leads to a contradiction.

🔹 7. Common Fallacies (Errors in Reasoning)

  • Affirming the Consequent:

    P ⇒ Q and Q is true, so P is true (❌ invalid)

  • Denying the Antecedent:

    P ⇒ Q and P is false, so Q is false (❌ invalid)

🧠 Example:


Statement 1 (Given):
All even numbers are divisible by 2. ✅

Statement 2 (Given):
8 is an even number. ✅

Conclusion (Using reasoning):
Therefore, 8 is divisible by 2. ✅

Why is this reasoning?

You're applying a general rule (all even numbers are divisible by 2) to a specific case (8 is an even number) to reach a logical conclusion (8 is divisible by 2).

This is a simple direct proof using deductive reasoning.

Great! Let's go through a few very basic examples of different types of mathematical reasoning:

🔹 1. Direct Proof (Deductive Reasoning)

Statement:

If a number is even, then it is divisible by 2.

Proof:


Let’s say we take the number 10.
10 is even because it can be written as 2 × 5.
Therefore, 10 is divisible by 2.

✅ Conclusion: The statement holds true.

🔹 2. Proof by Contrapositive

Statement:

If a number is not divisible by 2, then it is not even.

Contrapositive:

Instead of proving:

P ⇒ Q → If a number is even, then it is divisible by 2

We prove:

¬Q ⇒ ¬P → If a number is not divisible by 2, then it is not even

Example:


Take 7.
7 is not divisible by 2.
Therefore, 7 is not even.

✅ Contrapositive is true → So the original statement is also true.

🔹 3. Proof by Contradiction

Statement:

There is no smallest positive rational number.

Assume the opposite (for contradiction):

Suppose there is a smallest positive rational number, say 𝑟.

Now consider r/2​, which is also a positive rational number, and smaller than 𝑟

This contradicts our assumption that 𝑟 r is the smallest.

❌ So our assumption is false.

✅ Conclusion: There is no smallest positive rational number.

🔹 4. Inductive Reasoning

Statement (Pattern Guessing):

The sum of the first few odd numbers seems to form a pattern.

Let’s test:


1 = 1
1 + 3 = 4
1 + 3 + 5 = 9
1 + 3 + 5 + 7 = 16

Looks like:

1 → 1^2, 4 → 2^2, 9 → 3^2, 16 → 4^2

Guess (Induction):

Sum of first 𝑛 odd numbers = 𝑛^2

This is not a proof yet, but inductive reasoning helps us discover the pattern.

What is irrational number

An irrational number is a type of real number that cannot be expressed as a fraction of two integers — that is, it cannot be written in the form:

a/b

where 𝑎 a and 𝑏 b are integers and 𝑏 ≠ 0 b  =0.

Key Characteristics of Irrational Numbers:

  • Their decimal representation goes on forever and does not repeat.
  • They cannot be written exactly as a simple fraction.
  • 𝜋 π (pi) ≈ 3.1415926535…
  • √2≈ 1.41421356…
  • e (Euler's number) ≈ 2.718281828…

Contrast with Rational Numbers:

  • Rational: Can be expressed as fractions (e.g., 3/4, -2, 0.5)
  • Irrational: Cannot be expressed as fractions (e.g., √3, π)
