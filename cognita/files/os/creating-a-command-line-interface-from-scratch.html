Creating a CLI (Command Line Interface)

Creating a CLI (Command Line Interface) from scratch involves writing a program that can parse user input from the terminal, execute actions based on commands or flags, and provide helpful output. Here‚Äôs a clear step-by-step guide to creating one from scratch, using Python as an example language (though the same principles apply in most languages).

  • What commands or options will it support?

‚öôÔ∏è 3. Parse Arguments Manually (from Scratch)


import sys

def main():
    if len(sys.argv) < 2:
        print("Usage: mycli  [options]")
        return

    command = sys.argv[1]

    if command == "greet":
        name = "there"
        if "--name" in sys.argv:
            name_index = sys.argv.index("--name") + 1
            if name_index < len(sys.argv):
                name = sys.argv[name_index]
        print(f"Hello, {name}!")
    else:
        print(f"Unknown command: {command}")

if __name__ == "__main__":
    main()

Try:


./mycli.py greet --name Alice

üì¶ 4. Make it a Real CLI Tool (Optional)

You can install it locally using setup.py or pyproject.toml, or link it manually:


ln -s $(pwd)/mycli.py /usr/local/bin/mycli

Then run it from anywhere:


mycli greet --name Alice

üß∞ 5. Alternative: Use argparse or Libraries (Not From Scratch)

If you're open to using libraries:


import argparse

parser = argparse.ArgumentParser(description="A simple greeting CLI")
parser.add_argument("command", choices=["greet"])
parser.add_argument("--name", help="Name to greet", default="there")

args = parser.parse_args()

if args.command == "greet":
    print(f"Hello, {args.name}!")

‚úÖ CLI from Scratch: Key Takeaways

  • Use sys.argv to read arguments directly (for a truly "from scratch" approach).
  • Implement command parsing logic manually.
  • Add error handling, help messages, and validation for user-friendly output.

const args = process.argv.slice(2); // skip ['node', 'cli.js']

if (args.length === 0) {
  console.log("Usage: cli.js greet --name <name>");
  process.exit(1);
}

const command = args[0];

if (command === "greet") {
  const nameIndex = args.indexOf("--name");
  let name = "there";
  if (nameIndex !== -1 && nameIndex + 1 < args.length) {
    name = args[nameIndex + 1];
  }
  console.log(`Hello, ${name}!`);
} else {
  console.log(`Unknown command: ${command}`);
}
