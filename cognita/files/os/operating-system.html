Operating System (OS)

An Operating System (OS) is the core software that manages computer hardware and software resources and provides services for computer programs. It acts as an intermediary between users, applications, and the computer hardware.

üåê 1. What is an Operating System?

An Operating System is software that:

  • Controls and coordinates the use of hardware among various applications.
  • Provides a user interface (CLI or GUI).
  • Manages system resources like CPU, memory, disk, and I/O devices.

Examples: Windows, Linux, macOS, Android, iOS

You can build a simple OS in steps:

  1. Write a bootloader ‚Äì Loads your OS kernel from disk to RAM.
  2. Create a kernel ‚Äì With basic CPU and memory management.
  3. Set up interrupt handling ‚Äì For keyboard, timers, etc.
  4. Add a basic file system ‚Äì For reading/writing to disk.
  5. Build a shell or GUI ‚Äì So users can interact.

Tools used:

  • Languages: C, Assembly
  • Emulators: QEMU, Bochs
  • Build tools: GCC, NASM, Make
  • File Systems: FAT, ext2
