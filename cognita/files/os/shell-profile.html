Shell Profile

A shell profile refers to a configuration file that sets up your environment when a shell session starts. It's specific to the Unix/Linux/macOS command-line interface (CLI), and it defines things like:

  • Environment variables (e.g., PATH, EDITOR)
  • Aliases (e.g., alias ll='ls -la')
  • Custom functions or scripts
  • Shell behavior (e.g., prompt customization)
