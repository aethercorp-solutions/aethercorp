What is UIScreen in iOS?

UIScreen is a UIKit class that represents a physical screen (display) attached to the device. It provides information about the screen‚Äôs size, scale, brightness, and coordinate space.

Most iPhones and iPads only have one screen, but UIScreen becomes especially useful when dealing with:

  • External displays (e.g. AirPlay, HDMI)
  • Multi-screen setups

Access the main screen:


let screen = UIScreen.main

Get screen bounds:


let bounds = UIScreen.main.bounds

Get screen scale (for retina):


let scale = UIScreen.main.scale  // e.g., 2.0 or 3.0
Property Description
main The device‚Äôs primary screen.
bounds The screen's bounds in points.
scale The screen scale factor (points to pixels ratio).
nativeBounds The screen size in actual pixels.
nativeScale The actual scale factor for rendering.
brightness Current screen brightness (0.0 to 1.0).
mirrored Whether this screen mirrors another.
availableModes List of screen resolution modes (used for external displays).

üìê Coordinate System

  • Screen coordinates are in points, not pixels.
  • The origin (0,0) is usually at the top-left corner.
  • Use scale or nativeScale if you need pixel-perfect layouts.

You can detect and display UI on a second screen:


if let externalScreen = UIScreen.screens.last, UIScreen.screens.count > 1 {
    let externalWindow = UIWindow(frame: externalScreen.bounds)
    externalWindow.screen = externalScreen
    externalWindow.rootViewController = ExternalDisplayViewController()
    externalWindow.isHidden = false
}

Concept Description

UIWindow A container for views shown on a screen.
UIScene A logical unit of UI (can live in a window, tied to a screen).
