jonkykong/SideMenu Tutorial

SideMenu is a simple and highly customizable side menu library for iOS written in Swift.

1. Installation

Using CocoaPods

Add the following to your Podfile:


pod 'SideMenu'

Then run:


pod install

Using Swift Package Manager

  • In Xcode, go to File > Add Packages...
  • Enter the repository URL: https://github.com/jonkykong/SideMenu
  • Choose the version and add it to your project.

2. Basic Setup

  1. Import SideMenu

    In your view controller where you want to present the menu:

    
import SideMenu
  2. Create a Menu ViewController

    Create a new UIViewController subclass (e.g., MenuListController) that will be the content of the side menu.

    Example:

    
class MenuListController: UITableViewController {
    let items = ["Home", "Profile", "Settings"]
    
    override func tableView(_ tableView: UITableView, numberOfRowsInSection section: Int) -> Int {
        return items.count
    }
    
    override func tableView(_ tableView: UITableView, cellForRowAt indexPath: IndexPath) -> UITableViewCell {
        let cell = UITableViewCell()
        cell.textLabel?.text = items[indexPath.row]
        return cell
    }
    
    override func tableView(_ tableView: UITableView, didSelectRowAt indexPath: IndexPath) {
        // Handle menu item tap
        print("Selected: \(items[indexPath.row])")
        dismiss(animated: true, completion: nil)
    }
}
  3. Initialize and Present the SideMenu

    In your main view controller (the one with a button or gesture to open the menu):

    
class MainViewController: UIViewController {

    var menu: SideMenuNavigationController?

    override func viewDidLoad() {
        super.viewDidLoad()

        let menuVC = MenuListController()
        menu = SideMenuNavigationController(rootViewController: menuVC)
        
        // Set the presentation style
        menu?.leftSide = true // or false for right side

        // Optional: Customize the menu
        SideMenuManager.default.leftMenuNavigationController = menu
        SideMenuManager.default.addPanGestureToPresent(toView: self.view)
        
        // Add a button to open the menu
        navigationItem.leftBarButtonItem = UIBarButtonItem(title: "Menu", style: .plain, target: self, action: #selector(didTapMenu))
    }
    
    @objc func didTapMenu() {
        if let menu = menu {
            present(menu, animated: true)
        }
    }
}
  4. Customization

    You can customize the menu by changing properties on SideMenuNavigationController:

    
menu?.menuWidth = max(round(min((UIScreen.main.bounds.width), (UIScreen.main.bounds.height)) * 0.75), 240)
menu?.presentationStyle = .menuSlideIn
menu?.statusBarEndAlpha = 0
    • menuWidth controls the width of the menu.
    • presentationStyle can be .menuSlideIn, .viewSlideOut, .menuDissolveIn, etc.
    • statusBarEndAlpha controls the dimming of the status bar.
  5. Adding Gestures

    SideMenu allows you to add gestures to open the menu:

    
SideMenuManager.default.addPanGestureToPresent(toView: self.view)
SideMenuManager.default.addScreenEdgePanGesturesToPresent(toView: self.view)
  6. Handling Menu Selection

    In the menu's didSelectRowAt method, you can communicate back to the main view controller, e.g., via delegate, notifications, or closures.

Summary

  • Install SideMenu via CocoaPods or Swift Package Manager.
  • Create a menu view controller subclass.
  • Initialize SideMenuNavigationController with the menu controller.
  • Present it on button tap or gestures.
  • Customize appearance and presentation style.
