enum

An enum (short for enumeration) is a special data type used in many programming languages to define a set of named constant values. Enums are helpful when you want to represent a variable that can only take one out of a small set of possible values.

ðŸ”¹ Basic Syntax in Different Languages

C / C++


enum Color {
    RED,
    GREEN,
    BLUE
};

Color favoriteColor = GREEN;

Java


enum Color {
    RED, GREEN, BLUE;
}

Color favoriteColor = Color.GREEN;

C#


enum Color {
    Red,
    Green,
    Blue
}

Color favoriteColor = Color.Green;

Python (using the enum module)


from enum import Enum

class Color(Enum):
    RED = 1
    GREEN = 2
    BLUE = 3

favorite_color = Color.GREEN

TypeScript


enum Color {
    Red,
    Green,
    Blue
}

let favoriteColor: Color = Color.Green;

ðŸ”¸ Why Use Enums?

  • Readability: Named constants are easier to understand than numeric values.
  • Safety: You limit a variable to only certain values.
  • Maintainability: Easier to update or expand the set of values later.
