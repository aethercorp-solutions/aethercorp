‚úÖ 2. Control System Using Classes (MVC style)

A ‚Äúcontrol system‚Äù often means state + controller logic.

Example: a Temperature Controller.

Model (state)


@interface TemperatureModel : NSObject
@property (nonatomic) float temp;
@end

@implementation TemperatureModel
@end

Controller


@interface TempController : NSObject
- (void)adjust:(TemperatureModel *)model;
@end

@implementation TempController

- (void)adjust:(TemperatureModel *)model {
    if (model.temp > 40) {
        NSLog(@"Cooling system ON");
    } else if (model.temp < 20) {
        NSLog(@"Heating system ON");
    } else {
        NSLog(@"System stable");
    }
}

@end

Using it


TemperatureModel *m = [TemperatureModel new];
m.temp = 45;

TempController *c = [TempController new];
[c adjust:m];

‚úÖ 3. Event-based Control System (Target/Action or Delegates)

Control via Delegates (common in Cocoa Touch)

Example: Monitoring a value and notifying when it changes.


@protocol SpeedDelegate 
    - (void)speedDidChange:(int)newSpeed;
@end
    
@interface SpeedController : NSObject
@property (nonatomic, weak) id delegate;
@property (nonatomic) int speed;
@end

@implementation SpeedController

- (void)setSpeed:(int)speed {
    _speed = speed;
    [self.delegate speedDidChange:speed];
}

@end

‚úÖ 5. If you meant a REAL engineering control system (PID controller)

Objective-C can implement PID logic:


@interface PID : NSObject
@property float Kp, Ki, Kd;
@property float prevError;
@property float integral;

- (float)compute:(float)setpoint actual:(float)actual dt:(float)dt;
@end


@implementation PID

- (float)compute:(float)setpoint actual:(float)actual dt:(float)dt {

    float error = setpoint - actual;
    
    self.integral += error * dt;
    float derivative = (error - self.prevError) / dt;

    float output = self.Kp * error + self.Ki * self.integral + self.Kd * derivative;

    self.prevError = error;
    return output;
}
@end
