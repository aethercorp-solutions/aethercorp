🔥 What is an iOS LLM API?

An LLM API lets your iOS app send text (or multimodal data) to a Large Language Model and receive generated responses such as:

  • Text generation
  • Code generation
  • Summarization
  • Classification
  • Chat responses
  • Tool calling
  • Structured JSON outputs

Common providers:

  • OpenAI
  • Google (Gemini)
  • Anthropic (Claude)
  • Microsoft (Azure OpenAI)

3️⃣ Example: Adding AI Chat to iOS (Swift)

Step 1 – Backend Endpoint

POST /ai/chat

Request:


{
    "message": "Summarize this inventory list..."
}

Backend calls LLM → returns result.

Step 2 – Swift API Call


struct ChatRequest: Codable {
    let message: String
}

struct ChatResponse: Codable {
    let reply: String
}

func sendMessage(_ text: String) async throws -> String {
    let url = URL(string: "https://api.yourserver.com/ai/chat")!
    
    var request = URLRequest(url: url)
    request.httpMethod = "POST"
    request.setValue("application/json", forHTTPHeaderField: "Content-Type")
    
    let body = ChatRequest(message: text)
    request.httpBody = try JSONEncoder().encode(body)
    
    let (data, _) = try await URLSession.shared.data(for: request)
    let response = try JSONDecoder().decode(ChatResponse.self, from: data)
    
    return response.reply
}
