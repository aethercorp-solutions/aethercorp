Google People API

The Google People API is a service provided by Google that allows developers to access and manage information about people, such as contacts, profiles, and directory information. It's commonly used in applications that need to read or update user contacts, retrieve profile data, or interact with Google's contact management system.

Key Features of Google People API

  • Access user profile information: Retrieve information about authenticated users like their name, email, photo, etc.
  • Manage contacts: Read, create, update, and delete contacts.
  • Get contact groups: Retrieve contact groups and membership.
  • Directory information: Access Google Workspace directory information (if applicable).

Common Use Cases

  • Building contact management applications.
  • Syncing contacts between different platforms.
  • Accessing user profile info for personalization.
  • Creating CRM tools integrated with Google Contacts.

Basic Overview of How to Use Google People API

  1. Enable the API:
    • Go to Google Cloud Console.
    • Create or select a project.
    • Enable the People API for your project.
  2. Set up OAuth 2.0 credentials:
    • Create OAuth client ID credentials.
    • Set up scopes you need (e.g., https://www.googleapis.com/auth/contacts.readonly).
  3. Use the API:
    • Use Google API client libraries (available in multiple languages) or direct HTTP requests.
    • Request access tokens via OAuth 2.0.
    • Call endpoints like people.get, people.connections.list, etc.

Example: Get Profile Info with Python


from google.oauth2 import credentials
from googleapiclient.discovery import build

# Assume you have a valid access token with People API scope
creds = credentials.Credentials(token='YOUR_ACCESS_TOKEN')

service = build('people', 'v1', credentials=creds)

# Get profile info
profile = service.people().get(resourceName='people/me', personFields='names,emailAddresses').execute()

print(profile)

2. OAuth 2.0 Scopes

To use the API, your app must request appropriate scopes during authentication. Common scopes:

Scope Description
https://www.googleapis.com/auth/contacts Read/write access to contacts
https://www.googleapis.com/auth/contacts.readonly Read-only access to contacts
https://www.googleapis.com/auth/userinfo.profile Basic user profile info
https://www.googleapis.com/auth/user.emails.read Read user's email addresses

3. Authentication

Use OAuth 2.0 to get an access token from the user. Popular libraries:

  • Python: google-auth, google-auth-oauthlib
  • Node.js: googleapis npm package
  • JavaScript (client-side): Google Sign-In SDK

4. Making API Calls

Use the access token to call the API endpoints. You can use any HTTP client or Google SDKs.

ðŸ§ª Example: List Contacts (Python)


from google.oauth2.credentials import Credentials
from googleapiclient.discovery import build

creds = Credentials(token='YOUR_ACCESS_TOKEN')
service = build('people', 'v1', credentials=creds)

results = service.people().connections().list(
    resourceName='people/me',
    personFields='names,emailAddresses',
    pageSize=100
).execute()

for person in results.get('connections', []):
    name = person.get('names', [{}])[0].get('displayName')
    email = person.get('emailAddresses', [{}])[0].get('value')
    print(f"{name}: {email}")
