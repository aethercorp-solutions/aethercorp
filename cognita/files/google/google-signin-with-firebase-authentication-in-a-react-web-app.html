Google Sign-In with Firebase Authentication in a React web app

To implement Google Sign-In with Firebase Authentication in a React web app, follow this step-by-step guide:

üîß Prerequisites

  • Node.js & npm installed
  • A React project (create-react-app or Vite)
  • A Firebase project (console.firebase.google.com)

üõ†Ô∏è Step 1: Install Firebase


npm install firebase

üî• Step 2: Configure Firebase

Go to the Firebase Console:

  1. Create a project
  2. Add a web app to your Firebase project
  3. Copy the Firebase config

Enable Google Auth Provider:

  • Go to Authentication > Sign-in method
  • Enable Google under the sign-in providers

üìÅ Step 3: Setup Firebase in Your Project

Create a firebase.js file in src/:


// src/firebase.js
import { initializeApp } from "firebase/app";
import { getAuth, GoogleAuthProvider } from "firebase/auth";

const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  projectId: "YOUR_PROJECT_ID",
  appId: "YOUR_APP_ID",
  // ...other config
};

const app = initializeApp(firebaseConfig);
const auth = getAuth(app);
const provider = new GoogleAuthProvider();

export { auth, provider };

üß™ Step 4: Implement Google Sign-In in React

Create a component like Login.js:


// src/Login.js
import React from 'react';
import { signInWithPopup } from "firebase/auth";
import { auth, provider } from './firebase';

const Login = () => {
  const handleGoogleSignIn = async () => {
    try {
      const result = await signInWithPopup(auth, provider);
      const user = result.user;
      console.log("User Info:", user);
    } catch (error) {
      console.error("Google Sign-in Error:", error);
    }
  };

  return (
    <div>
        <h2gt;Login with Google</h2gt;
        <button onClick={handleGoogleSignIn}>Sign in with Google</buttongt;
    </divgt;
  );
};

export default Login;

üß† Step 5: Track Authentication State (Optional)

Use React state or Context to manage auth:


// src/App.js
import React, { useEffect, useState } from 'react';
import { onAuthStateChanged, signOut } from "firebase/auth";
import { auth } from './firebase';
import Login from './Login';

const App = () => {
  const [user, setUser] = useState(null);

  useEffect(() => {
    const unsubscribe = onAuthStateChanged(auth, (currentUser) => {
      setUser(currentUser);
    });
    return () => unsubscribe();
  }, []);

  const handleSignOut = () => {
    signOut(auth);
  };

  return (
    <div>
      {user ? (
        <div>
            <h2>Welcome, {user.displayName}</h2>
            <img src={user.photoURL} alt="profile" />
            <button onClick={handleSignOut}>Sign Out</button>
        </div>
      ) : (
        <Login />
      )}
    </div>
  );
};

export default App;
