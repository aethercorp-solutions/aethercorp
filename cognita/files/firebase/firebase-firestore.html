Firebase Firestore

Firebase Firestore is a NoSQL, cloud-hosted database provided by Google as part of the Firebase platform. It's designed to store, sync, and query data for web, mobile, and server development at scale.

ðŸ”¥ Key Features of Firestore:

  1. Cloud-based NoSQL Database
    • Stores data in documents, which are organized into collections.
    • Documents are like JSON objects (key-value pairs).
  2. Real-time Synchronization
    • Automatically syncs data across all clients in real time.
    • Great for chat apps, collaborative tools, etc.
  3. Offline Support
    • Built-in support for offline data access and sync.
    • Apps can continue functioning without network connectivity.
  4. Powerful Queries
    • Query collections with filters, ordering, and limits.
    • Indexed automatically for fast performance.
  5. Scalable
    • Firestore scales automatically to handle large data sets and high traffic.
  6. Security with Firestore Rules
    • Secure your data with Firebase Security Rules.
    • Define access control per document/collection level.
  7. Multi-region & Strong Consistency
    • Data is replicated across regions for high availability.
    • Offers strong consistency guarantees.

ðŸš€ Example Use (JavaScript/Node):


import { initializeApp } from "firebase/app";
import { getFirestore, collection, addDoc, getDocs } from "firebase/firestore";

const firebaseConfig = {
  // your config here
};

const app = initializeApp(firebaseConfig);
const db = getFirestore(app);

// Add a document
await addDoc(collection(db, "users"), {
  name: "John Doe",
  age: 25
});

// Read documents
const querySnapshot = await getDocs(collection(db, "users"));
querySnapshot.forEach((doc) => {
  console.log(`${doc.id} => `, doc.data());
});
