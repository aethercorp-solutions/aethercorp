Firebase Cloud Messaging (FCM)

Firebase Cloud Messaging (FCM) is a free service provided by Google Firebase that allows you to send push notifications and messages to users across platforms such as iOS, Android, and web applications.

  1. Cross-platform Messaging

  2. Targeted Messaging

    • To a specific device
    • Device groups
    • Topics (e.g., "news", "weather")
    • User segments

  3. Data Messages

    Send custom key-value pairs (invisible to the user) to trigger actions in the app.

  4. Notification Messages

    Automatically handled by the system to show messages in the system tray.

  5. Analytics & A/B Testing

    Integrated with Firebase Analytics for performance tracking and conversion metrics.
