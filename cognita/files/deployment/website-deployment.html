Website deployment

Website deployment involves making a website available on the internet so users can access it through a web browser. Here are some general terms commonly used in website deployment:

Development Terms

  • Frontend: The part of the website users interact with directly (HTML, CSS, JavaScript).
  • Backend: The server-side code and database handling the logic and data (e.g., PHP, Python, Node.js).
  • Build: The process of converting source code into a deployable format (e.g., minifying JavaScript, compiling SCSS).
  • Repository: A storage space for code, usually managed by Git (e.g., GitHub, GitLab, Bitbucket).

Hosting & Server Terms

  • Web Hosting: A service that stores your website files and makes them accessible online.
  • Server: A computer that delivers web pages to users' browsers.
  • Domain Name: The human-readable web address (e.g., example.com).
  • DNS (Domain Name System): Translates domain names into IP addresses.
  • IP Address: A unique address that identifies a server on the internet.
  • SSL Certificate: Provides secure HTTPS encryption.

Deployment Tools & Methods

  • FTP/SFTP: Protocols for transferring files to a web server.
  • CI/CD (Continuous Integration / Continuous Deployment): Automating code integration, testing, and deployment.
  • Static Site Generator: Tools like Hugo, Jekyll, or Next.js (for static builds).
  • Package Manager: Tools like npm or yarn to manage project dependencies.

Environment & Configuration

  • Environment Variables: Configurations that differ between environments (e.g., API keys).
  • Production: The live version of the website.
  • Staging: A test environment that mimics production.
  • Development Environment: Where developers write and test code before staging or production.

Performance & Monitoring

  • CDN (Content Delivery Network): Distributes content closer to users for faster load times.
  • Caching: Storing parts of the website to reduce load times.
  • Monitoring: Tracking uptime, performance, and errors (e.g., with tools like New Relic, Sentry).
