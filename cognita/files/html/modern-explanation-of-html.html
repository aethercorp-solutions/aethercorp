1. What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages.

  • It structures content on the web
  • It tells the browser what content is (not how it looks)
  • It works together with:
    • CSS ‚Üí design & layout
    • JavaScript ‚Üí behavior & interactivity

üëâ Think of HTML as the skeleton of a website.

2. What HTML Is Used For

HTML is used to:

  • Display text (headings, paragraphs)
  • Show images and videos
  • Create links and navigation
  • Build forms (login, signup, search)
  • Structure apps, dashboards, blogs, e-commerce sites

Every website you visit uses HTML.

3. How Browsers Use HTML

  1. Browser downloads the HTML file
  2. Parses the HTML into a DOM (Document Object Model)
  3. Loads linked CSS and JavaScript
  4. Renders the page on screen
  5. Executes JavaScript for interactions

HTML ‚Üí DOM ‚Üí CSS applied ‚Üí JS behavior ‚Üí Visible Page

5. Explanation of Each Part


<!DOCTYPE html>
  • Tells the browser to use HTML5
  • Required for modern standards

<html lang="en">
  • Root element of the page
  • lang helps accessibility and SEO

<head> (Metadata Section)

Not visible to users


<meta charset="UTF-8">
  • Supports all languages & emojis 

<meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Makes pages responsive on mobile 

<title>My First Website</title>
  • Text shown in browser tab

<body> (Visible Content)

Everything users see is inside <body>.

6. Semantic HTML (Very Important)

Semantic HTML uses meaningful tags instead of generic <div>.

Common Semantic Elements
<header> Top section
<nav> Navigation menu
<main> Main content
<section> Logical grouping
<article> Independent content
<aside> Side content
<footer> Bottom section
