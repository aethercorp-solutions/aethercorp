1. What HTML Is (Core Purpose)

HTML (HyperText Markup Language) is the structural language of the web.

It tells the browser:

  • What content exists (text, images, links, forms, videos)
  • What role each piece plays (heading, paragraph, button, input, nav)
  • How content is organized, not how it looks

HTML is:

  • ‚ùå Not a programming language
  • ‚úÖ A markup language (describes structure using tags)

Example:


<h1>Hello World</h1>
<p>This is a paragraph.</p>

2. How HTML Is Written (Usage)

HTML is written using elements (tags):

Basic HTML Element Anatomy

3. Basic HTML Document Structure

Every HTML page follows this structure:


<!DOCTYPE html>
<html>
    <head>
    <title>My Page</title>
    <meta charset="UTF-8">
    </head>
    <body>
    <h1>Welcome</h1>
    <p>Hello, world!</p>
    </body>
</html>

What Each Part Does

Section Purpose
<!DOCTYPE> Tells browser to use HTML5
<html> Root of document
<head> Metadata (not visible)
<body> Visible content

9. Step 3: Render Tree

Browser combines:

  • DOM
  • CSSOM

Creates Render Tree:

  • Only visible elements
  • With final computed styles

Hidden elements (display: none) are excluded

10. Step 4: Layout (Reflow)

Browser calculates:

  • Element width & height
  • Position on screen
  • Box model

This is expensive computationally.

Triggered by:

  • Window resize
  • Font change
  • DOM modifications

11. Step 5: Paint

Browser:

  • Draws pixels
  • Text, colors, borders, shadows

12. Step 6: Composite (GPU)

Browser layers are:

  • Sent to GPU
  • Combined efficiently

Used for:

  • Animations
  • Transforms
  • Opacity
