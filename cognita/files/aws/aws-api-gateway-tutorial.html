🌐 AWS API Gateway Tutorial (Beginner → Advanced)

Here’s a complete AWS API Gateway tutorial tailored for developers like you (Node.js + serverless + React Native experience), starting from fundamentals to production-level usage 🚀

🧠 What Is AWS API Gateway?

AWS API Gateway is a fully managed service to create REST, HTTP, or WebSocket APIs that:

  • Expose endpoints like /users, /login
  • Route requests to Lambda, EC2, containers, or external APIs
  • Handle auth, throttling, caching, monitoring

🏗️ Architecture Overview

Typical modern serverless architecture:


Client → API Gateway → Lambda → Database

Example stack:

  • React Native / Web app
  • API Gateway
  • AWS Lambda
  • DynamoDB / MongoDB

⚔️ API Types (Choose the Right One)

Type Use Case Cost Features
HTTP API Simple APIs 💸 Cheapest Fast, JWT auth
REST API Enterprise APIs 💰 Higher API keys, caching
WebSocket API Real-time ⚡ Chat, live updates

✅ Recommendation:

👉 Use HTTP API for modern apps unless you need advanced features.

🚀 Step-by-Step Tutorial (Hands-On)

1️⃣ Create a Lambda Function

Node.js example:


exports.handler = async (event) => {
    return {
        statusCode: 200,
        body: JSON.stringify({
        message: "Hello from API Gateway!"
        }),
    };
};

2️⃣ Create API Gateway (HTTP API)

AWS Console Steps

  1. Open API Gateway
  2. Click Create API
  3. Choose HTTP API
  4. Click Add Integration → Lambda
  5. Select your Lambda
  6. Click Create

3️⃣ Add Route

Example:


GET /hello

Attach Lambda integration.

4️⃣ Deploy API

  1. Click Deploy
  2. Copy the public URL

Test:


https://abc123.execute-api.us-east-1.amazonaws.com/hello

🎉 Done!

🔐 Add Authentication (JWT / Cognito)

Example with Cognito JWT Authorizer

Steps:

  1. Create Cognito User Pool
  2. API Gateway → Authorizers → JWT
  3. Attach authorizer to route

Client sends header:


Authorization: Bearer <JWT_TOKEN>

📦 Connect API Gateway to MongoDB (Lambda Example)


const mongoose = require("mongoose");

exports.handler = async () => {
  await mongoose.connect(process.env.MONGO_URI);

  return {
    statusCode: 200,
    body: JSON.stringify({ status: "connected" }),
  };
};

🧩 Deploy with Serverless Framework

Since you already use Serverless, here’s the fastest setup:

serverless.yml


service: my-api

provider:
  name: aws
  runtime: nodejs20.x

functions:
  hello:
    handler: handler.handler
    events:
      - httpApi:
          path: /hello
          method: GET

Deploy


serverless deploy
