Implement Firebase Analytics

To implement Firebase Analytics with an AWS serverless Node.js backend and a React Native app, the common architecture is:

Architecture

  • React Native App
    • Collects analytics events using Firebase SDK
    • Sends custom backend-tracked events to AWS Lambda if needed
  • AWS Serverless Backend
    • Receives API requests
    • Logs server-side analytics/events
    • Optionally forwards events to Firebase using GA4 Measurement Protocol

1. Setup Firebase Project

Go to:

Firebase Console

Create:

  • Android app
  • iOS app

Enable:

  • Google Analytics

Download:

  • google-services.json (Android)
  • GoogleService-Info.plist (iOS)

2. Install React Native Firebase Analytics

Official library:

React Native Firebase

Install:


npm install @react-native-firebase/app
npm install @react-native-firebase/analytics

iOS:


cd ios && pod install

3. Android Configuration

Add to:

android/build.gradle


classpath 'com.google.gms:google-services:4.4.0'

Add to:

android/app/build.gradle


apply plugin: 'com.google.gms.google-services'

Put:

  • google-services.json

inside:


android/app/

4. iOS Configuration

Add:

  • GoogleService-Info.plist

inside Xcode project.

5. Basic Analytics Tracking

App Open


import analytics from '@react-native-firebase/analytics';

await analytics().logAppOpen();

Screen Tracking


await analytics().logScreenView({
    screen_name: 'HomeScreen',
    screen_class: 'HomeScreen',
});

Custom Events


await analytics().logEvent('contact_created', {
    method: 'manual',
    platform: 'ios',
});

7. AWS Lambda Analytics Logging

Backend analytics are useful for:

  • API usage
  • Fraud detection
  • Feature usage
  • Admin dashboards
  • Billing
  • Server-side conversions

Example Lambda Handler


// handler.js

module.exports.trackEvent = async (event) => {
  try {
    const body = JSON.parse(event.body);

    console.log('Analytics Event:', body);

    return {
      statusCode: 200,
      body: JSON.stringify({
        success: true,
      }),
    };
  } catch (err) {
    return {
      statusCode: 500,
      body: JSON.stringify({
        error: err.message,
      }),
    };
  }
};

Serverless Framework Example


functions:
  trackEvent:
    handler: handler.trackEvent
    events:
      - http:
          path: analytics
          method: post
          cors: true

Deploy:


serverless deploy
