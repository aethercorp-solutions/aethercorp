firebase-admin

Firebase Admin SDK (commonly installed as firebase-admin) is a server-side library that lets you interact with Firebase services using admin privileges.

🔥 What is firebase-admin?

It’s the official SDK used in backend environments like:

  • Node.js servers
  • AWS Lambda (your case)
  • Cloud Functions

Unlike the client SDK (firebase), this one:

✅ Bypasses security rules

✅ Has full access to Firebase services

✅ Is meant for trusted environments only

📦 Install


npm install firebase-admin

🚀 Basic Usage

1. Import it


const admin = require('firebase-admin');

2. Initialize it


const serviceAccount = require('./serviceAccountKey.json');

admin.initializeApp({
  credential: admin.credential.cert(serviceAccount),
});

🔑 What You Can Do With It

🔐 Authentication (most common for you)


const decoded = await admin.auth().verifyIdToken(idToken);
console.log(decoded.uid);

👉 Used to verify users from your React Native app

🗄️ Firestore


const db = admin.firestore();
const users = await db.collection('users').get();

🔔 Push Notifications (FCM)


await admin.messaging().send({
    token: deviceToken,
    notification: {
        title: 'Hello',
        body: 'World',
    },
});

📁 Storage


const bucket = admin.storage().bucket();

⚠️ Important (Common Mistakes)

❌ Don’t use in frontend

  • Never use firebase-admin in React Native or web apps

❌ Missing credentials

Your earlier error:


serviceAccountKey is not defined

✔ Fix:


const serviceAccountKey = require('../config/firebase-service-account.json');

🧠 In Your Architecture (AWS Lambda + React Native)

Flow:

  1. User logs in (React Native using Firebase Auth)
  2. App gets idToken
  3. Sends token to your API
  4. Backend uses firebase-admin:

await admin.auth().verifyIdToken(idToken);

👉 This ensures the user is legit and authenticated

⚡ Pro Tip (Serverless Optimization)

In AWS Lambda, initialize outside the handler:


let adminApp;

if (!admin.apps.length) {
  adminApp = admin.initializeApp({
    credential: admin.credential.cert(serviceAccountKey),
  });
}

👉 Prevents re-initialization on every invocation
