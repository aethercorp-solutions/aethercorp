Production-ready AWS Serverless Node.js + Express API

Here’s a step-by-step guide to create a production-ready AWS Serverless Node.js + Express API using AWS Lambda + API Gateway + Serverless Framework.

🚀 1. Prerequisites

Install these first:

  • Node.js (LTS)
  • AWS CLI → configure it 
    
aws configure
  • Serverless Framework 
    
npm install -g serverless

📁 2. Create Project


mkdir serverless-express-api
cd serverless-express-api
npm init -y

Install dependencies:


npm install express serverless-http

Install dev dependencies:


npm install --save-dev serverless-offline
npm install express

🧠 3. Project Structure


serverless-express-api/
│── handler.js
│── app.js
│── serverless.yml
│── package.json

⚙️ 4. Create Express App (app.js)


const express = require('express');
const app = express();

app.use(express.json());

// Routes
app.get('/', (req, res) => {
    res.json({ message: 'Serverless Express API is running 🚀' });
});

app.get('/hello', (req, res) => {
    res.json({ message: 'Hello from Lambda + Express!' });
});

module.exports = app;

⚡ 5. Lambda Handler (handler.js)

This connects Express to AWS Lambda:


const serverless = require('serverless-http');
const app = require('./app');

module.exports.handler = serverless(app);

☁️ 6. Serverless Configuration (serverless.yml)


service: serverless-express-api

provider:
  name: aws
  runtime: nodejs20.x
  region: ap-southeast-1
  memorySize: 128
  timeout: 10

functions:
  api:
    handler: handler.handler
    events:
      - httpApi:
          path: /{proxy+}
          method: ANY

plugins:
  - serverless-offline

▶️ 7. Run Locally


serverless offline

Then open:


http://localhost:3000

Test:

  • /
  • /hello

🚀 8. Deploy to AWS


serverless deploy

After deployment you’ll get:


https://xxxx.execute-api.ap-southeast-1.amazonaws.com

🧪 9. Test Deployed API


curl https://xxxx.execute-api.ap-southeast-1.amazonaws.com/hello

🔐 10. Add Environment Variables (optional)

In serverless.yml:


provider:
  environment:
    DB_URL: mongodb+srv://...

Access in code:


process.env.DB_URL

🗄️ 11. Connect MongoDB (optional common setup)

Install:


npm install mongoose

Create db.js:


const mongoose = require('mongoose');

let conn = null;

module.exports.connect = async () => {
  if (conn) return conn;

  conn = await mongoose.connect(process.env.DB_URL);
  return conn;
};

Use inside routes:


const db = require('./db');

app.get('/users', async (req, res) => {
  await db.connect();
  res.json({ message: 'DB connected' });
});

⚡ 12. Production Best Practices

Add:

  • JWT authentication
  • API Gateway authorizers
  • CORS config
  • DynamoDB or MongoDB Atlas
  • CloudWatch logging
  • Separate dev/staging/prod stages
