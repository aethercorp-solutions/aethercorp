embedded C / microcontroller bit manipulation

That line is classic embedded C / microcontroller bit manipulation. Let’s break it down:

The code


PORTB |= (1 << 3);  // Set pin HIGH

What it does

It sets bit 3 of PORTB to 1, which typically means:

👉 Turn ON / set HIGH the pin PB3 (Pin 3 of Port B)

Step-by-step explanation

  1. (1 << 3)
    • This is a bitwise left shift
    • 1 << 3 = 00001000 (binary)
    • You're creating a mask where only bit 3 is 1
  2. PORTB |= ...
    • |= is a bitwise OR assignment
    • It keeps all existing bits the same except:
      • Bit 3 becomes 1

So effectively:


PORTB = PORTB | 0b00001000;

Result

  • Pin PB3 → HIGH (1)
  • Other pins in PORTB → unchanged

Why use this?

This is used in microcontroller programming (like Arduino, AVR, PIC, etc.) to:

  • Control GPIO pins directly
  • Avoid affecting other pins

Example (before & after)

Opposite operation (for reference)

To set the pin LOW:


PORTB &= ~(1 << 3);
