Bitwise operations

Bitwise operations are low-level tools that work directly on binary (0s and 1s). They’re extremely fast and commonly used in systems programming, embedded systems, networking, and performance-critical code.

Here are practical real-world use cases 👇

🔹 1. Flags & Permissions (VERY common)

Instead of using multiple booleans, you pack them into a single integer.

Example:


#define READ  1 << 0  // 0001
#define WRITE 1 << 1  // 0010
#define EXEC  1 << 2  // 0100

int permission = READ | WRITE; // 0011

Check permission:


if (permission & READ) {
    // has read access
}

👉 Used in:

  • File systems (Linux permissions)
  • User roles (admin, editor, viewer)
  • Feature toggles

🔹 2. Embedded Systems / Microcontrollers

You manipulate hardware registers directly using bitwise ops.

Example:


PORTB |= (1 << 3);  // Set pin HIGH
PORTB &= ~(1 << 3); // Set pin LOW

👉 Used in:

  • GPIO control (LEDs, sensors)
  • Communication protocols (SPI, I2C)
  • Bare-metal programming

🔹 3. Performance Optimization

Bitwise ops are faster than arithmetic operations.

Example:


x << 1  // same as x * 2
x >> 1  // same as x / 2

👉 Used in:

  • Game engines
  • Real-time systems
  • Competitive programming

🔹 4. Checking Odd or Even


if (x & 1) {
    // odd
} else {
    // even
}

👉 Faster than modulo % 2

🔹 5. Swapping Values (no temp variable)


a ^= b;
b ^= a;
a ^= b;

👉 Used in low-memory environments (though less common today)

🔹 6. Bit Masking (Extracting Data)

Used to isolate specific bits.

Example:


int value = 0b11010110;
int result = value & 0b00001111; // get lower 4 bits

👉 Used in:

  • Parsing network packets
  • Encoding/decoding protocols
  • Image processing

🔹 7. Networking / IP Addressing

Bitwise ops help calculate subnets.

Example:


network = ip & subnet_mask;

👉 Used in:

  • Routers
  • Firewalls
  • Backend systems handling IP filtering

🔹 8. Cryptography & Hashing

Bitwise ops are core to encryption algorithms.

👉 Used in:

  • XOR encryption
  • Hash functions (MD5, SHA)
  • Blockchain systems

🔹 9. Game Development

Compact state handling using bits.

Example:


#define JUMP  1 << 0
#define RUN   1 << 1

playerState |= JUMP;

👉 Used for:

  • Input systems
  • Physics flags
  • Collision states

🔹 10. Memory-efficient Data Storage

Store multiple values in one integer.

👉 Example:

  • 8 boolean flags in 1 byte instead of 8 bytes

🔥 When YOU (as a full-stack + embedded dev) will use it

Given your background:

  • AWS Lambda / Node.js → bit masks for permissions, feature flags
  • React Native → optimizing state flags or animations
  • Embedded / C/C++ → hardware control (most critical use)
  • Backend systems → encoding, compression, networking

⚡ Quick Mental Model

  • & → check
  • | → turn ON bits
  • ^ → toggle bits
  • ~ → invert bits
  • << → shift left (multiply)
  • >> → shift right (divide)
