Python machine-learning example

Here’s a clean, practical Python machine-learning example showing vector data representation, using something everyone recognizes: text → vectors → ML model.

We’ll go step by step so you can see what the vectors actually look like and why they matter.

Example: Text as vectors (Bag-of-Words / TF-IDF)

1. Raw data (not usable by ML yet)


documents = [
    "I love machine learning",
    "Machine learning is fun",
    "I hate bugs"
]

Models can’t work with strings — we need numerical vectors.

2. Vector representation using TF-IDF

Each document becomes a vector of numbers, one number per word.


from sklearn.feature_extraction.text import TfidfVectorizer

vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(documents)

print("Feature names (vector dimensions):")
print(vectorizer.get_feature_names_out())

print("\nVector representation:")
print(X.toarray())

Output conceptually looks like:


Features: ['bugs', 'fun', 'hate', 'learning', 'love', 'machine', 'is', 'i']

Document vectors:
[
 [0.0, 0.0, 0.0, 0.53, 0.53, 0.53, 0.0, 0.37],
 [0.0, 0.53, 0.0, 0.53, 0.0, 0.53, 0.53, 0.0],
 [0.57, 0.0, 0.57, 0.0, 0.0, 0.0, 0.0, 0.57]
]

👉 Each row = one document

👉 Each column = one word

👉 Each value = importance of that word

That matrix is the vector data representation.

3. Using vectors in a machine-learning model

Let’s classify sentiment (positive vs negative):


from sklearn.linear_model import LogisticRegression

y = [1, 1, 0]  # 1 = positive, 0 = negative

model = LogisticRegression()
model.fit(X, y)

# Predict new text
new_text = ["I love learning"]
new_vec = vectorizer.transform(new_text)

prediction = model.predict(new_vec)
print("Prediction:", prediction)

ML models never see text — they only see vectors like:


[0.0, 0.0, 0.0, 0.62, 0.62, 0.0, 0.0, 0.49]

4. Another common vector example: Numerical features

Vectors aren’t only for text.


# Each row is a vector: [height, weight, age]
X = [
    [170, 65, 30],
    [180, 80, 25],
    [160, 50, 40]
]

This is already vectorized — each sample is a feature vector in ℝ³.

5. Mental model (important 🧠)

Think of vectors as:

  • Points in space
  • Directions
  • Encoded meaning

In ML:

  • Text → vectors
  • Images → vectors
  • Audio → vectors
  • Tabular data → vectors

Everything becomes numbers.
